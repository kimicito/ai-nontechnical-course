第14级 — 销售与营销

14.2 外部任务：外部AI

⏱ 20分钟 · 面向营销人员、SMM经理、内容经理
公开数据（外部AI）
本节课的所有任务都涉及公开内容、开放来源、通用方法论。可以使用Claude、ChatGPT、Gemini。

为什么营销是使用外部AI最安全的领域之一

营销内容本质上是公开的。最终产品（帖子、宣传册、广告文案）面向广泛的受众。只要不泄露内部预算、价格和战略计划，商业机密泄露的风险极小。

重要：发送到云端之前，请检查请求中是否包含真实价格、预算、客户数据、战略计划。如果有，请去标识化或发送到本地部署。

1. 内容规划开发

生成创意、专栏、发布日历——这是生成式AI的天然领地。

提示词示例（Claude带网页搜索）
为B2B公司准备一个月的内容计划，行业为【请填写行业】：

参数：
- 渠道：LinkedIn、企业博客、邮件通讯
- 目标受众：【请填写：采购人员、工程师、总监】
- 主题板块：专业知识、案例、趋势、产品
- 频率：每周3篇LinkedIn帖子、每月2篇文章、4封邮件

请完成：
1. 每篇帖子的主题（标题 + 格式：轮播、文字、视频）
2. 发布日历
3. 每个主题的吸引点（什么能抓住注意力）
4. 行动号召（CTA）
5. 各渠道之间的交叉链接

效果：生成即用内容计划，节省4-5小时。

2. 广告活动文案撰写

生成标题变体、广告文案、描述——不含真实预算。

提示词示例
为B2B受众撰写5个广告文案变体：

产品：【请填写产品类型】，面向【请填写行业】
独特卖点：【请填写核心优势】
目标受众：【请填写职位】，【请填写公司规模】
格式：Google Ads（3个标题各30字符、2个描述各90字符）

额外要求：
- 3个LinkedIn变体（更专业的语气）
- 3个邮件通讯变体（个性化）
- A/B测试假设：测试什么（标题、CTA、优惠）

效果：一套测试广告文案，节省2-3小时。

3. 内容SEO优化

生成元描述、H1-H3标题、alt文本——公开内容。

提示词示例
优化文本以提升SEO：

原始文本（博客文章）：
【请粘贴文章文本】

关键词（优先级）：
1. 【请填写关键词1】
2. 【请填写关键词2】
3. 【请填写关键词3】

请完成：
1. H1标题（60字符，含关键词）
2. 元描述（150字符，含CTA）
3. H2-H3结构（含关键词嵌入）
4. 3张图片的alt文本
5. 内部优化检查清单（内链）

效果：优化后的文本，节省1.5小时。

4. 基于公开数据的竞争对手分析

产品线、定位、公开价格——这些信息都在公开渠道。

提示词示例（网页搜索）
对【请填写市场】细分市场中的3个竞争对手进行分析：

竞争对手：
1. 【请填写公司1】
2. 【请填写公司2】
3. 【请填写公司3】

需收集的数据（公开来源）：
- 产品线（类别、SKU）
- 定位（他们如何定位自己）
- 公开价格（如网站上有）
- 推广渠道（他们在哪里做广告）
- 优势（根据评论、案例）
- 劣势（投诉、空白点）

请完成：
1. 对比表
2. 定位地图（他们如何看待自己 vs 客户如何看待他们）
3. 空白点（竞争对手未覆盖的领域——我们的机会）
4. 差异化建议

效果：竞争对手概览，节省6-8小时。

5. 展会材料准备

展位演示文稿、宣传册文本、销售经理脚本——公开材料。

提示词示例
为【请填写展会名称】准备一套展会材料：

参数：
- 行业：【请填写】
- 受众：【请填写】
- 产品：【请填写】
- 展会目标：【请填写：线索、品牌、产品】

材料清单：
1. 展位口号（3个变体，5-7个字）
2. 宣传册文本（A4一页，结构：问题-解决方案-结果）
3. 销售经理脚本（问候 → 资格确认 → 演示 → CTA）
4. 易拉宝文本（2个变体，50字）
5. 展位见面后的跟进邮件

语气：专业，不咄咄逼人。

效果：一套完整材料，节省4-5小时。

6. 营销材料本地化

针对不同市场调整内容——语言、文化、法规。

提示词示例
将营销文本适配到本地市场：

原始文本（俄语）：
【请粘贴文本】

目标市场：【请填写国家】
语言：【请填写语言】

请适配：
1. 考虑行业术语的翻译
2. 文化细节（什么在这个市场有效）
3. 法规限制（什么不能承诺）
4. 示例和案例（如可能，替换为本地案例）
5. CTA（适合该市场的行动号召）

格式：表格"原文 → 适配 → 备注"

效果：本地化文本，节省3-4小时。

7. 行业趋势分析

汇总公开分析、新闻、分析师报告。

提示词示例（网页搜索）
分析【请填写】行业2025-2026年的趋势：

来源（公开）：
- 行业媒体
- 分析师报告（McKinsey、BCG、Deloitte）
- 大型企业的动态新闻
- LinkedIn上专家的文章

问题：
1. 关键趋势（前5）
2. 技术变革
3. 法规变化
4. 客户行为变化
5. 新商业模式
6. 对我司的风险和机会
7. 定位建议

格式：分析备忘录（1-2页）

效果：趋势概览，节省5-6小时。

8. 基于交易所报价的价格分析

监控公开交易所数据用于定价策略。

提示词示例（网页搜索）
对原材料市场进行价格分析：

指标：
1. LME铝（3个月期货）— 6个月动态
2. 波罗的海干散货指数（BDI）— 运费趋势
3. 美元/人民币汇率（如相关）
4. 制造业PMI指数

问题：
- 原材料价格走势如何？
- 这对【请填写产品】的成本有何影响？
- 下季度预测（根据分析师）
- 定价策略建议（通用，不含具体数字）

格式：分析简报

效果：市场概览，节省2-3小时。

知识检查

问题1：为什么内容计划开发适合外部AI？

因为最终产品（帖子、文章）本质上是公开的
因为不包含商业价格
只要不指定广告预算就可以

问题2：竞争对手分析属于哪个范围？

外部AI — 产品和价格的公开数据
本地部署 — 商业情报
混合 — 数据在外部，结论在内部

问题3：可以在外部AI中准备投资者推介材料吗？

可以，如果是不含财务数据的一般结构
混合 — 结构文本在外部，最终数字在本地部署
不行，推介材料总是包含内部信息
