1.4 · 文档处理：分析、摘要、对比

🎯 无代码 · ✅ 交互式测验

⚠️ 本课程中的示例仅用于教学目的。 上传合同、估算、证书及其他机密文件时， 必须使用 GENAI（企业级数据保护解决方案）， 而非开放的云服务（ChatGPT、Claude、Kimi）。

🎯 为什么企业需要这个

管理者平均每天阅读2-3小时文档：合同、报告、商务提案、验收单、会议纪要。

AI将其缩短到15-20分钟：

  • 合同摘要：30分钟 → 2分钟
  • 对比2份提案：1小时 → 5分钟
  • 风险分析：2小时 → 10分钟

总计节省：每天3-4小时

📚 学习内容

3种文档处理任务

1. 摘要（压缩）

用几个要点概括文档内容。

💡 示例提示词
你是律师。
用5点概括合同：
1. 采购内容
2. 价格
3. 交货期限
4. 付款条件
5. 违约金和罚款

如果某点未在合同中提及 — 写「未注明」。

节省： 30分钟 → 2分钟

2. 对比

按指定标准对比2份或更多文档。

💡 示例提示词
你是采购员。
按标准对比2份商务提案：
| 标准 | 提案1 | 提案2 | 更优 |
| 单价 | | | |
| 最小订单 | | | |
| 交货期 | | | |
| 付款条件 | | | |
| 质保 | | | |

高亮差异。给出推荐。

节省： 1小时 → 5分钟

3. 风险分析

在文档中查找潜在问题。

💡 示例提示词
你是有15年经验的公司法务。
分析供货合同中的风险。

每个风险注明：
- 描述
- 级别：高 / 中 / 低
- 降低建议

检查章节：
☐ 违约金
☐ 不可抗力
☐ 终止
☐ 争议解决程序
☐ 向第三方转让权利

节省： 2小时 → 10分钟

🛠️ 如何应用

场景：合同分析

步骤1： 将合同上传到AI（ChatGPT Plus、Claude Pro或Kimi支持文件上传）

步骤2： 指定角色和任务

完整提示词
你是公司法务律师。
分析上传的供货合同。

任务：
1. 提炼关键条款（价格、期限、数量、付款）
2. 找出买方风险
3. 评估每项风险：高 / 中 / 低
4. 给出风险降低建议
5. 指出哪些条款不常见或与标准不同

格式：风险表格 + 合同总体可接受性结论。

步骤3： 检查结果（尤其是风险评估）

步骤4： 保存提示词模板供未来合同使用

场景：对比商务提案

任务： 从3份商务提案中选择供应商。

提示词
你是资深采购员。
对比附件中的3份商务提案。

对比标准：
1. 单价（元）
2. 最小订单（件）
3. 生产周期（天）
4. 交货期（天）
5. 付款条件
6. 是否有认证
7. 质保期

以表格输出。每项标准标明领先者。
最后 — 明确推荐及理由。

限制：如果周期 > 14天 — 标记为风险。

👨‍💻 如何委派给开发人员

任务模板

"需要自动合同分析系统。

要求：
1. 文档上传：PDF、DOCX、TXT
2. 按模板自动分析：
   - 摘要（5个关键点）
   - 与公司模板对比
   - 风险评估（表格）
3. 结果保存到 [Notion/SharePoint/1C]
4. 通知负责人
5. 分析历史（按交易对手搜索）

示例输入和输出文档（附件）。

集成：[邮件 / 1C / CRM]
期限：[日期]"

✅ 结果验收检查清单

  • 合同所有关键条款都已考虑？
  • 风险评估正确？（检查2-3个点）
  • 建议可执行？（可以实施）
  • 格式便于传递给同事？
  • 没有幻觉？（事实与文档一致）
  • 关键合同 — 已由律师检查？

🎯 课程要点

AI在几分钟内分析文档：摘要、对比、发现风险。但始终要复核关键结论。AI是助手，不是律师的替代品。节省：每天3-4小时阅读文档时间。
