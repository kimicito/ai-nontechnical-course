1.5 · 实践：自动化你的第一个任务

🎯 实践 · ✅ 无代码

🎯 为什么企业需要这个

没有实践的理论无用。选择你工作中一个真实任务，并在1小时内将其自动化。

目标： 本课后你将拥有一个可工作的提示词，用于真实任务，每周节省2-3小时。

📚 步骤1：选择任务

适合自动化的理想任务

  • ✅ 每周重复 > 3次
  • ✅ 每次耗时 > 20分钟
  • ✅ 有明确算法（可描述步骤）
  • ✅ 不需要关键决策（出错安全）

按角色的任务示例

🛒 采购员

"对比3份商务提案并给出推荐"

"签署前检查合同风险"

"从CSV生成周采购报告"

📊 分析师

"分析月度销售并发现异常"

"基于趋势编制下季度预测"

⚖️ 法务

"检查5份合同的典型风险"

"将新合同与公司模板对比"

📦 物流

"优化10个配送点路线"

"计算不同方式配送成本"

🛠️ 步骤2：自动化算法

步骤1：描述任务

做什么、为什么、多久一次、每次耗时多少。

示例
任务：对比供应商商务提案
频率：每周5次
时间：每次1.5小时
问题：耗时，容易遗漏重要细节

步骤2：编写提示词（RCOF）

角色、上下文、目标、格式。

示例
你是一位经验丰富的FMCG采购员。
正在为饼干选择纸板包装供应商。
收到3份提案。

任务：按标准对比提案并推荐最佳方案。

标准：
1. 单价（含运费）
2. 最小订单量
3. 生产周期
4. 付款条件

格式：表格 + 带理由的推荐。
限制：周期 > 14天 = 风险。

步骤3：用3个示例验证

取3个真实示例，检查提示词是否有效。

检查内容
  • 结果符合请求？
  • 格式方便？
  • 没有错误/幻觉？
  • 时间缩短了？

步骤4：记录流程

在Notion / Word / Excel中创建模板。

模板结构
任务名称：[做什么]
频率：[多久一次]
优化前时间：[X分钟]
优化后时间：[Y分钟]

提示词：
[插入提示词]

输入示例：
[插入示例]

结果示例：
[插入结果]

验证日期：[日期]
是否有效：是 / 否（需修正）

步骤5：与同事分享

展示你如何节省时间。这会激励其他人。

如何分享
  • 在企业聊天中发送提示词模板
  • 在会议上展示："我将X时间缩短了"
  • 为团队创建带模板的知识库

📝 理解测验

问题1： 哪种任务适合用AI自动化？

A) 一年做一次的一次性任务
B) 每周重复>3次，每次>20分钟，有明确算法
C) 无明确标准的创意任务

问题2： 提示词中的RCOF是什么？

A) 模型名称
B) 角色、上下文、目标、格式 — 提示词结构
C) 文件字节大小

问题3： 验证提示词需要多少示例？

A) 至少3个真实示例
B) 1个示例足够
C) 无需验证，AI永远正确

📝 实践任务

任务：60分钟内自动化一个任务

  1. 0-10分钟： 从上方列表选择任务（或自己的）
  2. 10-25分钟： 按RCOF公式编写提示词
  3. 25-40分钟： 用3个真实示例验证
  4. 40-50分钟： 优化提示词（什么不工作？）
  5. 50-60分钟： 记录模板并保存

验收标准

  • 提示词包含RCOF
  • 已用3个示例验证
  • 结果稳定（请求间不变化）
  • 节省时间 > 50%
  • 模板已记录并可重复使用

🎯 课程要点

最好的学习方式是动手做。选择一个任务，编写提示词，用3个示例验证，记录流程。每周节省2-3小时是现实，不是理论。与同事分享结果 — 这会加速公司AI的应用。
