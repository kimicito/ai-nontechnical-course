0.3 · 云端 vs 本地部署：用于机密数据的生成式AI

🎯 无代码 · ✅ 末尾有测验

🎯 为什么企业需要这个

60%的公司因担心数据泄露而不使用AI。但在「把所有数据发送到ChatGPT」和「不使用AI」之间，还有第三条路 — 本地部署。

⚠️ 真实案例： 采购经理将供应商价格列表上传到ChatGPT。3个月后，竞争对手知道了他们的定价策略。数据通过模型训练泄露了。

📚 云端 vs 本地部署：简单解释

☁️ 云端 (Cloud)

这是什么： 您将数据发送到其他公司的服务器（OpenAI、Anthropic、Google）。

✅ 便宜、快速，无需IT部门

❌ 数据离开公司

🏢 本地部署 (On-premise)

这是什么： AI模型在公司内部服务器上运行，数据不会离开办公室。

✅ 完全控制数据，保密性

❌ 更贵，需要服务器和管理员

谈判类比

💡 类比

云端 = 在商业中心租办公室。 方便，但门房能看到所有访客，邻居隔着墙能听到。

本地部署 = 自己的房子。 您是主人，但自己修水管和付屋顶维修费。

🛠️ GENAi：中间方案

GENAi 是一个企业AI平台（类似ChatGPT，但仅供公司内部使用）。员工看到熟悉的聊天界面，但数据留在公司内部。您总能在Rusal Online主页上找到「GENAi」按钮。

何时选择GENAi：

  • 处理商业机密（价格、供应商、合同）
  • 员工或客户个人数据
  • 发布前的财务报告
  • 战略计划和预测
  • 监管要求数据存储在境内（俄罗斯、中国、欧盟）

风险矩阵

🔴 高风险

合同、价格、并购

🟡 中风险

内部报告、人力资源

🟢 低风险

营销、公开数据

要求：
- 用户数量：[N]人
- 文档：每次最多[N]页
- 保密性：关键（商业机密）
- 语言：RU, EN
- 硬件预算：[金额]
- 期限：[N]天

请提供：
1. 模型方案（Qwen / Llama / Mistral）
2. 服务器配置
3. 许可证成本
4. 备份方案
5. 用户操作指南"

✅ 结果验收检查清单

  • 数据不会离开公司服务器？（检查防火墙设置）
  • 有备份方案？（如果服务器烧毁）
  • 用户无需IT专家也能操作？（界面易于理解）
  • 响应速度 < 10秒？（峰值负载测试）
  • 有模型更新流程？（新版本每月发布）

🧰 工具

📝 理解测验

问题1： 可以用ChatGPT分析供应商合同吗？

A) 可以，ChatGPT对任何数据都安全
B) 不行，这是商业机密。需要本地部署（GENAi + Qwen/Llama）
C) 可以，只要删除历史记录

问题2： 什么是GENAi？

A) Google的付费聊天机器人
B) 本地模型的免费界面。数据不会发送到互联网
C) 云端数据存储服务

问题3： 本地部署LLM需要什么预算？

A) 服务器约$2000-5000（或GPU约$100-500/月）+ 管理员20%时间
B) 免费，笔记本就够了
C) 只需$20/月，像ChatGPT Plus一样

🎯 课程要点

公开数据用云端模型（ChatGPT、Claude）。机密数据用本地部署（GENAi + Qwen/Llama）。没有通用方案 — 根据数据类别选择。切勿将商业机密上传到公共聊天机器人。
