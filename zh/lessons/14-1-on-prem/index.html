第14级 — 销售与营销

14.1 内部任务：本地部署

⏱ 25分钟 · 面向销售经理、商务总监、营销人员
封闭环境（本地部署）
本节课的所有任务均涉及真实价格、交易条件、客户数据、商业预测。请使用内部模型或本地大语言模型。

为什么本地部署对销售至关重要

销售是采购的镜像功能：在采购中您是买方，拥有自己的谈判立场；而在销售中您是卖方，数据敏感性是对称的。价格、折扣、付款条件——无论从交易的哪一方来看，都是商业秘密。

此外，需求预测和销售计划直接影响生产计划，因此这项任务比乍看之下更为敏感。

必须本地部署的场景：
  • 定价和计算个性化折扣（谈判立场的基础）
  • 销售合同的准备和审核（真实交易条件）
  • 客户群分析——细分、潜力、流失（客户依赖性）
  • 需求预测和销售计划（内部运营数据）
  • 与关键客户的谈判准备（谈判策略）
  • CRM记录和通话摘要（个人数据 + 商业信息）
  • 回复客户的RFQ/RFP（提交前的真实价格）
  • 具体交易的输赢分析（揭示客户流失原因）
  • 销售计划执行报告（管理财务信息）
  • 基于真实内部指标的活动效果分析（预算、ROI）
  • 正式披露前的投资者/分析师推介（内幕信息）

1. 定价和计算个性化条件

针对特定客户计算价格，考虑数量、折扣、付款条件和物流——属于商业秘密。

提示词示例（本地部署）
为[请填写客户名称]准备商业报价：

基础数据：
- 产品：[请填写产品]
- 数量：[请填写数量] 单位/月
n- 基础价格：[请填写价格] 美元/单位
- 交易历史：[请填写折扣历史]
- 竞争对手：报价[请填写竞争对手价格]
- 付款条件：[请填写条件]
- 物流：送货至[请填写地址]

请计算：
1. 考虑数量折扣的价格
2. 方案：预付款 vs 延期付款（对价格的影响）
3. 与竞争对手的比较（定位）
4. 最低价格（盈利能力的下限）
5. 谈判策略建议

成果：有据可依的价格和下限，节省2–3小时。

2. 客户群分析

细分、增长潜力评估、流失风险——战略信息。

提示词示例
分析客户群并给出建议：

数据（匿名化/内部）：
- 客户A：营业额[请填写]，利润率[请填写]，逾期0天，年增长+15%
- 客户B：营业额[请填写]，利润率[请填写]，逾期45天，保持稳定
- 客户C：营业额[请填写]，利润率[请填写]，逾期0天，下降-20%/年
- 客户D：新客户，潜力[请填写]，行业[请填写]

请完成：
1. ABC细分（按营业额和利润率）
2. 流失风险评估（按趋势）
3. 每个细分领域的增长潜力
4. 建议：保留谁、发展谁、放弃谁
5. 季度行动计划

成果：带计划的细分，节省4–5小时。

3. 需求预测和销售计划

基于现成数据的文本推理——预测直接影响生产计划。

提示词示例
准备2026年第四季度的需求预测和销售计划：

销售历史（内部数据）：
- Q1：[请填写数量] 单位，收入[请填写]
- Q2：[请填写数量] 单位，收入[请填写]
- Q3（预测）：[请填写数量] 单位

影响因素：
- 季节性：第四季度通常比Q3+[请填写]%
- 新客户：漏斗中有[请填写数量]个
- 风险：[请填写竞争因素]
- 营销活动：[请填写计划]

请完成：
1. 按月的数量和收入预测
2. 客户计划（前10名）
3. 风险和缓解措施
4. 对生产/库存的要求

成果：有据可依的预测，节省3–4小时。

4. 与关键客户的谈判准备

谈判策略、BATNA、杠杆——如同采购，只是坐在桌子的另一边。

提示词示例
准备与[请填写客户名称]的谈判策略：

背景：
- 当前合同：[请填写条件]，到期日[请填写日期]
- 数量：[请填写] 单位/年
- 客户投诉：[请填写诉求]
- 竞争对手：[请填写对手]提供[请填写条件]
- 我方立场：[请填写目标]

请制定：
1. BATNA（我方最佳替代方案）
2. ZOPA（可能达成协议的区域）
3. 压力杠杆（我们能提供什么/我们面临什么风险）
4. 情景：乐观、现实、悲观
5. 对典型反对意见的回应
6. 谈判计划（分阶段）

成果：现成的策略，节省3小时。

5. CRM记录和会议摘要

自动处理通话记录和会议记录——个人数据 + 商业信息。

提示词示例
根据笔记总结与客户的面谈：

基础数据（CRM + 经理笔记）：
- 日期：[请填写]
- 参与者：[请填写客户姓名、职位]
- 主题：讨论合同[请填写编号]的条件
- 关键要点：
  * 客户希望将付款期限从30天延长到60天
  * 客户对[请填写]批次质量不满意
  * 客户正在考虑竞争对手[请填写]
  * 客户准备在价格降低5%的情况下增加20%的采购量

请完成：
1. 简要摘要（3句话）
2. 已达成协议清单
3. 未解决问题
4. 风险和危险信号
5. 下一步行动（谁、什么、何时）
6. 给管理层的建议

成果：结构化的CRM记录，节省30分钟。

6. 回复招标询价（RFQ/RFP）

准备询价回复——提交前的真实价格和条件。

提示词示例
准备回复[请填写客户名称]的RFQ：

客户要求：
- 产品：[请填写]
- 数量：[请填写] 单位
- 交货条件：[请填写国际贸易术语]
- 期限：[请填写]
- 质量要求：[请填写标准]
- 回复格式：[请填写要求]

我方数据：
- 成本：[请填写]
- 最低利润率：[请填写]%
- 生产能力：[请填写]
- 当前负荷：[请填写]%

请制定：
1. 技术方案
2. 商业条件（价格、折扣、付款）
3. 交货时间表
4. 相对于竞争对手的优势
5. 风险和限制

成果：现成的招标回复，节省4–5小时。

7. 输赢分析

分析交易输赢原因——极度敏感的信息。

提示词示例
对交易[请填写编号]进行输赢分析：

交易数据：
- 客户：[请填写名称]
- 产品：[请填写]
- 数量：[请填写]
- 我方价格：[请填写]
- 获胜竞争对手价格：[请填写]（如已知）
- 结果：[赢/输]
- 客户反馈：[请填写]

请完成：
1. 交易时间线（阶段、参与者）
2. 决策因素（价格、质量、期限、关系）
3. 我方优势
4. 我方劣势
5. 竞争对手做得不同的地方
6. 教训和建议（流程、产品、价格）
7. 改进行动计划

成果：结构化分析，节省3小时。

8. 投资者推介（发布前）

包含最终数据的IR材料，在官方披露前——内幕信息。

提示词示例
基于内部数据准备投资者推介结构：

数据（初步，非公开）：
- 上半年收入：[请填写]（增长[请填写]%）
- EBITDA：[请填写]
- 新客户：[请填写]
- 丢失交易：[请填写]
- 年度预测：[请填写]
- 关键风险：[请填写]

请制定：
1. 执行摘要（3页）
2. 财务结果（关键指标）
3. 运营成就
4. 下半年战略
5. 风险和缓解措施
6. 对复杂问题的回答（分析师可能会问什么）

语气：专业、自信、不夸大。

成果：现成的推介，节省5–6小时。

知识测验

问题1：为什么需求预测需要本地部署，而不是外部AI？

因为它直接影响生产计划，包含运营数据
因为它包含产品价格
仅当预测包含超过3年的历史数据时

问题2：可以在外部AI中进行输赢分析吗？

可以，如果匿名化客户名称
不行，它揭示了客户流失给竞争对手的原因——极度敏感
仅针对与大客户输掉的交易

问题3：与客户会面后的CRM记录应该使用哪种部署方式？

外部AI——这只是文本摘要
本地部署——联系人个人数据 + 商业信息
混合模式——匿名化后发送到云端
