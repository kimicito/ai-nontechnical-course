第12级 — 设备维护（维修与保养）

12.2 设备维护的外部任务

⏱ 15分钟 · 面向可靠性工程师和机械师
开放数据（外部AI）
本课程的所有任务都使用公开来源、通用方法论和开放市场。可以使用Claude、ChatGPT、Gemini。

为什么云端比本地部署更适合设备维护

当需要以下功能时，云端模型更具优势：

  • 网络搜索：监控设备维护标准、设备国标、行业要求
  • 翻译：处理国外备件目录、ISO、API标准
  • 生成：从零创建培训材料、标准规程
  • 公开数据分析：LME金属价格、设备价格、对标分析
  • 寻找替代品：进口替代、备件交叉编号
重要：发送到云端之前，请检查请求中是否包含具体的资产编号、停机数据、工厂财务指标。如果有，请脱敏处理或发送到本地部署系统。

1. 计划性检修图表的编制与更新

计划性预防检修的通用方法论 — 不绑定特定工厂。

提示词示例（外部AI）
为冶金企业编制计划性检修图表的方法论：

1. 按重要性对设备进行分类（A — 关键设备，B — 重要设备，C — 辅助设备）
2. 按设备类型确定技术维护和维修周期
   - 小修：每月/每季度
   - 中修：每年1-2次
   - 大修：按运行时间或日历期限
3. 资源预留（班组、备件、设备）
4. 非计划故障时调整图表的方法
5. 执行控制KPI（计划完成率、停机时间）
6. 文件形式（图表、工单、验收单）

不包含具体的资产编号和工厂数据。

结果：适用于企业实施的方法论，节省4-5小时。

2. 通用设备维护规程的编制

典型设备（泵、磨机、压缩机）的文件结构 — 不绑定特定工厂。

提示词示例（Claude带网络搜索）
为卧式离心泵（通用型，输送水/矿浆）编制设备维护规程结构：

规程结构：
1. 适用范围（泵类型、运行条件）
2. 技术维护类型及周期（日常巡检、TO-1、TO-2、TO-3、大修）
3. 每种维护类型的工作清单（检查点）
4. 允许参数标准（振动、温度、流量）
5. 维修后验收标准
6. 记录形式（巡检日志、验收单）
7. 附件（润滑图、密封图）

不包含具体的资产编号和工厂数据。

结果：完整的规程结构，节省2-3小时。

3. 操作说明书 — 通用部分

设备操作的基本原则，如果没有工厂特定的改装。

提示词示例
为板式换热器（通用型）编制操作说明书：

章节：
1. 用途和工作原理
2. 技术参数（通用范围）
3. 人员要求
4. 启停程序
5. 运行中的控制参数（温度、压力、压差）
6. 预防性检查（检查内容、检查周期）
7. 典型故障及处理（不绑定具体案例）
8. 安全要求

语言：简洁明了，面向操作员和机械师。

结果：完整的说明书，节省3-4小时。

4. 维修人员的培训材料

基本原则、安全、标准程序 — 不绑定工厂的实际事故。

提示词示例
为机械师准备"旋转设备维修安全"培训模块：

格式：10页幻灯片（每页文字内容）
内容：
1. 危险类型（机械、电气、化学）
2. LOTO（上锁挂牌）—— 断电程序
3. 个人防护装备
4. 测试题（5道题附答案）

不包含工厂的具体案例 — 仅通用原则。

结果：完整的培训模块，节省4小时。

5. 设备和备件替代品搜索（进口替代）

开放市场 — 搜索交叉编号、替代品、替换件。

提示词示例（网络搜索）
寻找SKF 22320 EK/C3轴承的进口替代方案：

要求：
- 俄罗斯制造商或欧亚经济联盟国家
- 相同尺寸：d=100，D=215，B=73 mm
- 类型：调心滚子轴承，铜保持架
- 允许载荷：不低于原装

输出表格：制造商、型号、尺寸、参考价格、库存情况（如有数据）。

结果：采购用替代品清单，节省1-2小时。

6. 行业设备维护实践对标（RCM、TPM）

开放方法论 — 冶金、选矿、能源领域的最佳实践。

提示词示例（网络搜索）
对冶金行业设备维护实践进行对标分析（2024-2026）：

问题：
1. 哪些大型冶金企业实施了RCM（以可靠性为中心的维护）？
2. 宣称取得了哪些成果（停机减少、成本节约）？
3. 使用哪些IT系统进行设备维护（SAP PM、1C MRO、Infor）？
4. 应用哪些KPI指标（MTBF、MTTR、OEE）？
5. 2026年有哪些冶金设备维护会议？

制作带来源的结构化报告。

结果：行业实践概述，节省3-4小时。

7. RCM/TPM转型方法论开发

通用框架 — 公开方法，不绑定特定工厂。

提示词示例
为冶金企业准备TPM（全员生产维护）实施计划：

阶段（按月划分，12个月计划）：
1. 准备：人员培训、组建TPM团队
2. 自主维护（操作员执行基础保养）
3. 计划性技术维护（图表、规程）
4. 计划性维修（大修、中修）
5. 培训与发展（培训、认证）
6. 早期项目管理（采购时考虑可靠性）
7. 安全与质量（5S、目视化管理）

每个阶段：目标、成功指标、典型问题。

结果：完整的实施计划，节省5-6小时。

8. 招标文件准备 — 通用章节

维修工作技术任务书的标准章节 — 不包含具体交易的商务条件。

提示词示例
准备设备大修技术任务书的标准结构：

章节：
1. 总则（适用范围、规范性文件引用）
2. 对象技术描述（不包含具体的资产编号）
3. 工作清单（标准类型：缺陷检测、维修、测试）
4. 质量要求（验收标准、测试）
5. 文件要求（验收单、证书、照片）
6. 质保义务（标准期限）
7. 安全要求
8. 承包商报告形式

具体数据的插入位置：[请确认生产信息]。

结果：标准技术任务书模板，节省2小时。

因为供应商是外部合作伙伴
