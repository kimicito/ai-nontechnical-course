15.2 外部任务：外部AI

⏱ 20分钟 · 面向律师、合规专员
公开数据（外部AI）
本课所有任务涉及标准模板、公开立法、公开司法判决、培训材料。可以使用 Claude、ChatGPT、Gemini。

可以在外部AI中做什么

与涉及具体案件和交易对手的任务不同，与公开法律领域打交道既安全又高效：

  • 按模板编制标准合同（不含真实条款）
  • 立法变更监控（公开来源）
  • 司法实践分析（公开判决）
  • 专利检索（公开数据库）
  • 培训材料（通用性质）
  • 法律实践对标（公开数据）
核心规则：如果请求中没有公司名称、金额、具体日期、个人数据、战略规划——可以使用外部AI。

1. 标准合同模板

编制不含真实条款的标准合同结构是公开的方法论。

提示词示例（外部AI）
准备一份标准供货合同：

参数（不含具体条款）：
- 类型：货物供应合同
- 法律体系：[填写国家]
- 买方：大型企业（B2B）
- 供应商：生产商
- 标的：[填写商品类型，不含数量/价格]

请编制：
1. 前言（标准版）
2. 合同标的（描述结构）
3. 价格与结算方式（标准措辞）
4. 交货与风险转移（Incoterms——标准版）
5. 质量与验收（标准条款）
6. 双方责任（不含具体罚款）
7. 不可抗力（标准条款）
8. 争议解决程序（标准选项）
9. 最终条款

格式：每条——基础文本+备选方案

结果：法务部门的标准模板，节省3–4小时。

2. 立法变更监控

按具体主题汇总立法变更。

提示词示例（网络搜索）
分析过去6个月的立法变更：

主题：
1. [填写领域：公司法/税务/劳动法/反垄断]
2. [填写第二主题]
3. [填写第三主题]

司法管辖区：[填写国家/地区]

请完成：
1. 关键变更清单（日期、文件编号、核心内容）
2. 对业务的影响分析（通用，不针对特定公司）
3. 生效日期
4. 企业一般需采取的行动
5. 准备建议

格式：分析报告（1–2页）

结果：变更概览，节省4–5小时。

3. 司法实践分析

检索和分析类似问题的公开司法判决。

提示词示例（网络搜索）
分析以下问题的司法实践：

问题：[填写法律问题，不针对具体案件]
司法管辖区：[填写]
期间：最近2年

请完成：
1. 相关案例精选（5–10个判决）
2. 每个案例的简要内容
3. 法院立场（关键结论）
4. 实践趋势（法院做法的变化）
5. 对企业的普遍风险
6. 风险最小化建议

注意：仅限公开判决，若非公开案件不得提及具体公司名称。

结果：实践概览，节省5–6小时。

4. 专利检索与自由实施分析

检索公开专利数据库，分析自由实施空间。

提示词示例（网络搜索）
对以下技术进行专利检索：

技术：[填写领域，不含产品具体细节]
关键词：[填写3–5个术语]
IPC分类号：[如已知请填写]
司法管辖区：[填写：俄罗斯、美国、欧洲、中国等]

请完成：
1. 检索相关专利（5–10项）
2. 每项专利分析（保护范围、期限、状态）
3. 自由实施分析（是否存在障碍）
4. 规避建议（如有）
5. 后续行动建议

注意：此为初步分析，不能替代专利代理人。

结果：初步分析，节省3–4小时（但不能替代专利代理人）。

5. 员工培训材料

准备通用法律主题的培训材料。

提示词示例
为员工准备培训材料：

主题：[填写：商业秘密/反腐败/竞争/个人数据]
受众：[填写：全体员工/经理/销售人员]
格式：演示文稿（10页）+测试（5题）

请编制：
1. 培训目标（员工应掌握什么）
2. 核心条款（通俗易懂）
3. 情境示例（假设性，非真实案例）
4. 行为准则（该做与不该做）
5. 违规责任
6. 测试（5道选择题）
7. 自查清单

语气：易懂，避免法律术语。

结果：现成的培训材料，节省3–4小时。

6. 行业法律实践对标

基于公开数据与同类公司比较法律流程。

提示词示例（网络搜索）
对行业法律实践进行对标：

行业：[填写]
对标参数：
- 法务部门结构
- 法律科技工具使用
- 外部律师 vs 内部法务
- 诉讼负荷（典型争议）
- 法律服务成本（公开数据）
- 合规实践

来源：行业报告、出版物、会议

请完成：
1. 行业3–5家公司的实践概览（公开数据）
2. 对比表
3. 最佳实践（best practices）
4. 法务职能优化建议

注意：仅限公开数据，不得涉及具体合同金额。

结果：行业实践概览，节省4–5小时。

外部任务汇总表

任务所需输入输出格式节省时间
标准模板合同类型、法律体系完整模板3–4小时

知识检验

问题1：可以在外部AI中编制含具体价格的供货合同吗？

可以，如果是标准合同
不行，具体价格是商业秘密，应使用本地部署
可以，但仅限小型公司

问题2：司法实践分析——哪个回路？

外部AI——司法判决是公开的
本地部署——司法实践是保密的
混合——匿名化公司名称

问题3：专利检索——哪个回路？

外部AI——专利数据库是公开的
本地部署——这是公司知识产权
混合——数据库在外部，分析在内部
