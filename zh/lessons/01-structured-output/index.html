1.2 · 如何让AI以您需要的格式回答

🎯 无代码 · ✅ 交互式测验

🎯 为什么企业需要这个

AI默认以纯文本回答。您会得到一段文字，需要手动重新输入到Excel或Word中。

问题： 经理收到文本形式的供应商分析，花15分钟将其转移到表格中。

解决方案： 直接要求AI以表格形式输出结果。节省：15分钟 × 20个任务 = 每周5小时。

📚 学习内容

管理者使用的4种回答格式

📊 表格

用于对比、分析、报告。一键复制到Excel。

☑️ 勾选清单

用于检查、审计、行动计划。清晰直观。

📧 现成邮件

AI直接为供应商、客户或管理层写邮件。

📋 分步计划

明确的行动顺序，包含日期和负责人。

如何要求表格

💡 提示词示例
你是一位财务分析师。
分析3个投资项目（数据如下）。

以表格形式输出结果：
| 项目 | 回报期 | 风险 | 建议 |

表格后附简要说明。

如何要求勾选清单

💡 提示词示例
你是一位审计师。
检查供应合同的风险。

以清单形式输出结果：
☐ 风险：[描述] — 级别：[高/中/低] — 措施：[建议]

最后：总体结论 — 是否签署合同。

🛠️ 如何应用

场景1：对比供应商

任务： 从3个方案中选择包装供应商。

提示词
你是一位FMCG采购员。
按标准对比3个供应商并以表格输出：
| 标准 | 供应商A | 供应商B | 供应商C | 领先者 |
| 单价 | | | | |
| 最小订单 | | | | |
| 交货期 | | | | |
| 付款条件 | | | | |

给出建议：选择谁以及原因。

结果： 可直接用于管理层汇报的现成表格。一键复制到Excel。

场景2：合同审计

任务： 签署前检查合同风险。

提示词
你是一位企业律师。
分析供应合同的风险。

以清单形式输出结果：
☐ 风险：[描述] — 级别：[高/中/低] — 建议：[措施]

最后：总体结论 — 合同是否可接受。

场景3：撰写邮件

任务： 向供应商写质量投诉邮件。

提示词
你是一位采购经理。
以商务风格写一封给供应商的邮件。

背景：最近一批次次品率15%，超过可接受的3%。

格式：现成的商务风格邮件。
结构：
1. 引用合同和可接受的次品率
2. 事实（批次、日期、数量、附件照片）
3. 要求：更换或退款
4. 回复期限：5个工作日

语气：坚定但礼貌。不含威胁。

结果： 可立即发送的现成邮件。只需插入具体信息即可。

场景4：行动计划

任务： 制定新流程实施计划。

提示词
你是一位项目经理。
制定CRM系统实施计划。

格式：带日期的分步计划：
1. 步骤 — 期限 — 负责人 — 结果
2. 步骤 — 期限 — 负责人 — 结果
...

总共8-10个步骤。
实施周期：3个月。
预算：不超过50万元。

结果： 可提交管理层审批的现成计划。可复制到Word或Excel。

👨‍💻 如何委派给开发人员

任务模板

"需要一个工具，自动将AI的回答转换为Excel表格。

要求：
1. 粘贴AI文本 — 获得现成的.xlsx表格
2. 适用于任何表格（供应商对比、报告、计划）
3. 保留格式（粗体标题、颜色）
4. 简单界面：复制粘贴 — 完成
5. 用户无需编程

输入示例（附件）：
[插入AI回答示例]

预期结果（附件）：
[插入期望的Excel]

期限：[日期]"

✅ 结果验收检查清单

  • 格式符合任务要求？（表格 / 清单 / 邮件 / 计划）
  • 所有数据都在？（无遗漏）
  • 可复制到Excel / Word / Outlook？
  • 结构清晰无需解释？
  • 计量单位已注明？（货币、天数、百分比）
  • 抽查2-3项验证正确性？
