1.2 · 结构化输出：JSON、表格、检查清单

🎯 无代码 · ✅ 交互式测验

🎯 为什么企业需要这个

AI可以以任何格式输出结果。但默认是连续文本，需要手动复制到Excel或1C。

问题： 经理收到文本格式的分析，花15分钟整理成表格。

解决方案： 直接要求AI输出表格。节省：15分钟 × 20个任务 = 每周5小时。

📚 学习内容

4种输出格式

📊 表格

对比、分析、报告。易于复制到Excel。

📋 JSON

面向开发人员和自动化。结构化数据。

☑️ 检查清单

用于检查、审计、质量控制。

📝 Markdown

用于文档、说明、文章。到处可读。

如何要求表格

💡 示例提示词
你是一位财务分析师。
分析3个投资项目（数据如下）。

以表格格式输出结果：
| 项目 | 投资回报率 | 回收期 | 风险 | 建议 |

表格后附简要建议说明。

如何要求检查清单

💡 示例提示词
你是一位ISO 9001审计师。
按以下标准审计采购流程。

以检查清单格式输出结果：
☐ 标准1 — [状态：通过/未通过] — [备注]
☐ 标准2 — [状态] — [备注]

最后：总体合规性评估（百分比）。

🛠️ 如何应用

场景1：对比供应商

任务： 从3个选项中选择包装供应商。

提示词
你是FMCG采购员。
按标准对比3个报价并输出表格：
| 标准 | 供应商A | 供应商B | 供应商C | 领先者 |
| 单价 | | | | |
| 最小订单 | | | | |
| 交货期 | | | | |
| 付款条件 | | | | |

给出建议：选择谁及原因。

结果： 可直接复制到演示文稿的现成表格。一键复制到Excel。

场景2：合同审计

任务： 检查合同中的风险。

提示词
你是企业法务。
分析供货合同中的风险。

以检查清单格式输出：
☐ 风险：[描述] — 级别：[高/中/低] — 建议：[怎么做]

最后：合同总体可接受性结论。

场景3：周报

任务： 准备周报。

提示词
你是商业分析师。
根据以下数据准备周报。

格式：
1. 关键指标（表格：指标 | 数值 | 环比变化）
2. 前3趋势（列表）
3. 建议（带优先级的列表）
4. 风险（表格：风险 | 概率 | 影响）

👨‍💻 如何委派给开发人员

任务模板

"需要解析AI回答到标准化格式。

要求：
1. 自动识别格式（表格/JSON/检查清单）
2. 将表格转换为Excel (.xlsx)
3. 将JSON转换为CSV
4. 检查数据完整性（无空字段）
5. 集成API端点

示例输入（附件）：
[插入AI回答示例]

预期结果（附件）：
[插入期望的Excel/CSV]

期限：[日期]"

✅ 结果验收检查清单

  • 格式符合要求？（表格 / JSON / 检查清单）
  • 所有字段已填写？（无空缺）
  • 数据正确？（随机检查2-3个单元格）
  • 可复制到Excel / 1C / CRM？
  • 结构无需解释即可理解？
  • 注明了计量单位？（元、天、百分比）
