第14级 — 销售与营销

14.3 混合任务

⏱ 20分钟 · 面向销售人员和营销人员
混合方案
在外部AI中准备结构和方法论。
真实数据（价格、客户、条件）添加到本地部署中。

销售与营销混合方案的核心原则

关键规则：数据越敏感（具体价格、交易条件、客户个人数据）——越要严格使用本地部署。通用方法论（文档结构、典型场景）可以安全地放在外部范围。

云端去标识化规则：
必须移除可以保留
具体价格和折扣定价方法论
客户名称和联系方式客户类型（大客户、中型、关键客户）
个别付款条件典型付款形式
具体交易数量数量计算方法论
公司财务指标行业基准
具体交易日期季节性、典型周期

1. 商业报价（混合）

云端：通用模板和报价结构。
本地部署：最终价格、条件、客户个性化。

步骤1：云端 — 报价模板
为B2B销售准备商业报价结构：

报价结构：
1. 封面（标志、联系方式、日期）
2. 公司简介（2-3句话）
3. 理解客户需求
4. 产品/解决方案（描述、规格）
5. 交付条件【请填写：具体条件】
6. 价格【请填写：按客户计算】
7. 优势和保障
8. 工作流程（阶段、时间）
9. 附件（规格、证书）

每个部分写什么、常见错误、表述示例。

【请填写：客户行业、产品类型、地区】
步骤2：本地部署 — 按客户填写
为【请填写客户名称】填写报价：

客户数据：
- 行业：【请填写】
- 消费量：【请填写】
- 合作历史：从【请填写年份】起合作，营业额【请填写】
- 当前条件：【请填写】
- 需求：【请填写】

我方数据：
- 价格：【请填写】（基础），量达到【请填写】时折扣【请填写】%
- 付款条件：【请填写】
- 交付时间：【请填写】
- 保障：【请填写】

编制最终个性化报价。

节省：云端模板（1.5小时）+ 本地部署填写报价（1小时）。

2. 谈判准备（混合）

云端：通用谈判方法论、典型场景。
本地部署：真实数字、具体客户、BATNA。

步骤1：云端 — 方法论
为B2B销售准备谈判方法论：

谈判阶段：
1. 侦察（提哪些问题以了解需求）
2. 方案演示（如何将产品与需求关联）
3. 处理异议（典型异议及回应）
4. 条件讨论（价格、折扣、付款、物流）
5. 成交（哪些信号表明客户准备就绪）

方法论：
- BATNA：如何确定和使用
- ZOPA：如何扩大协议区域
- 杠杆：我们可以提供什么（除价格外）
- 典型陷阱（过早折扣、无条件同意）

【请填写：行业、产品类型、平均客单价】
步骤2：本地部署 — 具体交易
将方法论应用于与【请填写客户】的谈判：

背景：
- 当前合同：【请填写】，到期【请填写日期】
- 数量：【请填写】单位/年
- 投诉：【请填写】
- 竞争对手提供：【请填写条件】
- 我方目标：【请填写】

请编制：
1. BATNA（如果无法达成协议）
2. ZOPA（可接受范围）
3. 杠杆（我们可以提供什么）
4. 场景：乐观、现实、悲观
5. 预期异议的回应

节省：云端方法论（1.5小时）+ 本地部署策略（1.5小时）。

3. 向潜在客户发送邮件（混合）

云端：通用邮件模板、A/B变体。
本地部署：用客户真实数据进行个性化。

步骤1：云端 — 邮件模板
为B2B冷邮件编写模板：

参数：
- 产品：【请填写类型】
- 目标受众：【请填写职位】，【请填写公司规模】
- 目标：获得会面/演示

变体：
1. 简短版（3句话，移动端格式）
2. 中等版（段落 + 要点 + CTA）
3. 带案例链接（社会证明）

每个版本：
- 邮件主题（3个变体）
- 吸引点（什么能抓住注意力）
- CTA（具体行动）
- 签名

【请填写：行业、目标受众核心痛点】
步骤2：本地部署 — 个性化
为【请填写姓名】，【请填写职位】，【请填写公司】个性化邮件：

客户数据：
- 行业：【请填写】
- 近期事件：【请填写：扩张、新产品、发布】
- 可能的需求：【请填写】
- 互动历史：【请填写】

模板：【请粘贴云端模板】

请调整：
1. 提及具体事件/背景
2. 将产品与他们的需求关联
3. 添加相关案例
4. 提出具体的会面时间

节省：云端模板（1小时）+ 本地部署个性化（30分钟）。

4. 投资者关系材料（混合）

云端：通用演示结构、典型幻灯片。
本地部署：正式发布前的最终数据。

步骤1：云端 — 演示结构
为B2B公司准备IR演示结构：

幻灯片：
1. 封面（公司、期间）
2. 执行摘要（3个关键信息）
3. 市场定位（TAM/SAM/SOM、趋势）
4. 产品线（类别、差异化）
5. 财务结果【请填写：具体数字】
6. 客户群（多元化、留存率）
7. 增长战略（有机增长、并购、新市场）
8. ESG和可持续发展
9. 风险与缓解措施
10. 财务预测【请填写：具体数字】
11. 投资论点（为什么值得投资）
12. IR部门联系方式

每张幻灯片：关键信息、分析师常见问题、如何回答。
步骤2：本地部署 — 最终数据
用最终数据填充IR演示：

初步数据（非公开）：
- 上半年收入：【请填写】
- EBITDA：【请填写】
- 利润率：【请填写】%
- 新客户：【请填写】
- 丢失交易：【请填写】
- 年度预测：【请填写】

请编制：
1. 每张幻灯片的文本（含数字）
2. 演讲备注（口头说什么）
3. 预期问题的回答
4. 附录（给分析师的详细分解）

节省：云端结构（2小时）+ 本地部署填写（2小时）。

知识检查

问题1：商业报价的混合原则是什么？

通用模板在云端，最终价格和条件在本地部署
如果去标识化客户名称，整个报价都可以在云端完成
整个报价只能在本地部署，因为包含商业秘密

问题2：向潜在客户的邮件发送——本地部署负责哪一步？

个性化：姓名、公司、背景、具体优惠
编写通用邮件模板
邮件变体的A/B测试

问题3：以下哪项发送到云端前必须移除？

具体价格、客户名称、交易数量
产品类型（例如"软件"）
客户行业（例如"制造业"）
