第12级 — 设备维护（维修与保养）

12.1 设备维护内部任务

⏱ 20分钟 · 面向可靠性工程师和机械师
封闭环路（本地部署）
本课的所有任务均与具体设备、实际故障、财务数据相关。请使用内部模型或本地大语言模型。

为什么本地部署对设备维护很重要

设备维护数据不仅仅是备件清单。它涉及：

  • 真实资产的状态 — 缺陷报告描述了具体机器的磨损情况
  • 生产脆弱性 — 故障历史揭示了生产线的薄弱环节
  • 财务义务 — 完工报告，是向承包商付款的依据
  • 竞争信息 — 停机时间、维修计划、预算
本地部署为必需的情况：
  • 具体设备的缺陷报告
  • 基于实际故障数据的根因分析
  • 含工厂物料清单的备件规格
  • 故障历史和平均故障间隔时间（MTBF，内部可靠性数据）
  • 维修文件（报告、记录）— 涉及财务
  • 与供应商的往来函件（商业条款）
  • 索赔工作（涉及法律）
  • 具体设备的振动诊断/无损检测解读
  • 面向管理层的可靠性报告
  • 停机及原因分析（可能涉及人员）
  • 维修预算论证
  • 「维修 vs 更换」论证 — 投资决策
  • 大修承包商报价分析

1. 制定和更新计划性预防维修（PPM）时间表

计划性预防维修（PPM）是规划的基础。大语言模型帮助根据实际数据更新时间表。

提示词示例（本地部署）
请更新3号车间2026年第三季度的PPM时间表：

原始数据：
- 当前时间表：磨机M-3大修（8月），压缩机中修（7月、9月）
- 实际情况：压缩机K-05于2026-07-10故障，需进行非计划维修
- 资源：机械班组6人，可用率80%
- 限制条件：8月为休假高峰期，2人休假

请完成：
1. 根据K-05非计划维修调整时间表
2. 在7月和9月之间重新分配工作
3. 评估每月班组工作量（人/时）
4. 风险分析：如再出现非计划任务会有什么影响
5. 产能储备建议

结果：修正后的PPM时间表，节省2–3小时。

2. 编制缺陷报告

设备检查后，将发现的缺陷整理成标准文件。

提示词示例（本地大语言模型）
根据电机ED-37（3号车间）检查数据：
- 非驱动端轴承振动：8.2 mm/s（标准≤4.5）
- 绕组温度：127°C（标准≤105）
- 轴承润滑油：变色，含杂质
- 轴窜动量：0.35 mm（标准≤0.1）

请按格式编制缺陷报告：
1. 设备名称、资产编号
2. 发现的缺陷（清单）
3. 严重程度评估（严重 / 重大 / 轻微）
4. 建议措施
5. 整改期限
6. 责任部门

结果：完整的缺陷报告，节省30分钟。

3. 故障根因分析（RCA，5个为什么）

针对具体设备的故障原因调查。

提示词示例
故障：泵P-12（煅烧车间渣浆输送）于2026-07-15 03:20停机。

数据：
- 自上次维修后的运行时长：4,847小时
- 前次故障：2025-11-03，更换轴承
- 停机前当前参数：振动12.4 mm/s，温度89°C
- 目视检查：叶轮损坏，轴套划痕

请进行根因分析：
1. 5个为什么（5层追问）
2. 石川图（6M：人、机、料、法、环、测）
3. 故障主要原因
4. 纠正措施（短期）
5. 预防措施（长期）
6. 维修计划变更建议

结果：结构化调查报告，节省2–3小时。

4. 备件规格准备

针对具体维修，列出所需材料和备件并关联物料清单。

提示词示例
需要对减速器G-08（磨机M-3传动）进行大修。

根据缺陷检查结果：
- 轴承22320（2件）— 磨损，需更换
- 中间轴 — 轴颈磨损（需堆焊+磨削）
- 圆柱齿轮 — 齿面划痕（需更换）
- 油封80×100×10（4件）
- 润滑油I-40A — 45升（冲洗+加注）

请准备备件和材料规格：
1. 序号、名称、图纸/国标
2. 数量、计量单位
3. 企业物料编码（如有）
4. 推荐供应商（来自批准供应商库）
5. 预估单价
6. 交货期
7. 标识：库存 / 需订购

请补充更换备注（例如：如原厂件不可用，可用替代品）。

结果：用于采购的完整规格，节省1小时。

5. 故障历史和平均故障间隔时间（MTBF）分析

计算平均无故障运行时间并识别规律。

提示词示例
压缩机K-05（准备车间）故障历史：
- 2024-03-12：吸气阀故障，运行3,200小时
- 2024-09-08：轴承过热，运行2,850小时
- 2025-02-20：密封件损坏，运行2,100小时
- 2025-08-15：连杆断裂，运行2,400小时
- 2026-01-30：排气阀磨损，运行2,650小时
- 2026-07-10：轴承过热（重复），运行2,200小时

请分析：
1. 期间MTBF（平均故障间隔时间）
2. MTBF趋势（改善 / 恶化 / 稳定）
3. 常见故障（哪些部件最频繁）
4. 维修计划变更建议
5. 预测：下次故障预计时间

结果：用于规划的统计分析，节省1.5小时。

6. 大修技术任务书编制

技术任务书是承包商的主要文件，包含工作范围、技术要求和工期。

提示词示例（本地部署）
请编制磨机M-3（资产编号 M-03-2012）大修技术任务书：

技术数据：
- 投产年份：2012，累计运行：87,400小时
- 上次大修：2021年
- 型号：球磨机3.2×15米，产能120吨/时
- 传动：减速器G-08，电机800 kW

工作清单（根据缺陷检查结果）：
1. 更换衬板（全套，约45吨）
2. 减速器G-08维修/更换
3. 润滑系统更换（泵、管道）
4. 端部密封维修
5. 试运转（空载、带载）
6. 验收标准（振动≤4.5 mm/s，温度≤65°C）

要求：
- 工期：21个日历日
- 质保期：18个月
- 文件：隐蔽工程记录、材料合格证、照片报告

结果：用于招标的完整技术任务书，节省3–4小时。

7. 维修文件（报告、记录）

完成工作的文件编制 — 向承包商付款的依据。

提示词示例
请编制变压器T-14大修完工报告：

完成工作：
- 更换绝缘油（30,000升）
- 油箱维修（焊接、密封）
- 更换高压套管（6件）
- 试验（击穿电压、介质损耗角正切tg δ）
- 外壳喷漆

承包商：LLC "EnergoRemont"
合同编号 45/2026，日期 2026-01-15
工期：2026-07-01 — 2026-07-14

格式：按标准表格编制的报告，含附件（隐蔽工程、试验记录）。

结果：完整的报告，节省45分钟。

8. 与供应商和服务公司的往来函件

协调备件交付条件、服务期限等。

提示词示例
请为以下询证准备致供应商LLC "TekhnoSnab"的函件：

主题：确认减速器G-08用轴承22320的交货期

背景：
- 订单于2026-06-20下达（申请单号 Z-1847）
- 承诺交期：4周
- 当前状态：尚未到货，时间紧迫（维修计划于2026-08-05开始）
- 备选方案：同型号SKF轴承（需确认兼容性）

语气：商务、紧迫但不冲突。提及可能按合同收取违约金。

结果：完整的函件，节省20分钟。

9. 质保索赔工作

就质保期内的故障向供应商或承包商提出索赔。

提示词示例
请编制质保索赔函：

设备：风机VO-16，供应商 LLC "VentProm"
合同：编号 78/2025，日期 2025-03-10
质保期：18个月
投运日期：2025-06-15

故障：叶轮叶片断裂（2026-07-08）
原因（初步评估）：铸造缺陷（固定区域存在气孔）
故障时运行时长：8,420小时（在质保期内）

索赔要求：
1. 免费更换叶轮
2. 赔偿拆装费用
3. 赔偿停机损失（2个班次）

附件：检查记录、缺陷照片、实验室结论。

结果：符合法律规范的索赔函，节省1–1.5小时。

10. 振动诊断和无损检测报告解读

对现成报告进行文本分析 — 不能替代信号处理/机器学习，但有助于形成结论表述。

提示词示例
请解读泵P-12的振动诊断报告：

数据：
- 测点：非驱动端轴承，径向振动
- 数值（mm/s）：总振值 — 8.2，1× — 5.1，2× — 1.8，3× — 0.4，BPF — 0.3
- 3个月趋势：从4.8上升至8.2 mm/s
- 轴承温度：67°C（标准≤65）

请给出结论：
1. 频谱显示什么（主导频率）
2. 可能原因（不平衡 / 轴承故障 / 汽蚀 / 其他）
3. 建议（紧急程度、措施）
4. 预测：多少小时后达到临界水平

结果：结构化的解读报告，节省30分钟。

11. 面向管理层的可靠性报告

设备状态和管理趋势的综合报告。

提示词示例
请编制3号车间2026年6月设备可靠性月度报告：

数据：
- 设备总数：147台
- 故障次数：12次（5月为8次）
- 计划维修：5次
- 非计划停机：7次
- 总停机时间：34小时
- 平均MTBF：2,340小时（5月：2,680小时）
- 严重故障：2次（压缩机K-05、泵P-12）

格式：1页A4纸，面向生产总监。
结构：关键指标、趋势、问题区域、建议。

结果：完整的报告，节省1小时。

12. 停机及原因分析

按类型、设备、原因对停机进行分类 — 用于决策。

提示词示例
分析3号车间2026年第二季度停机情况：

数据（小时）：
- 设备：泵 — 28，压缩机 — 45，输送机 — 12，磨机 — 38
- 原因：磨损 — 67，操作失误 — 18，外部因素 — 14，其他 — 24
- 班次：白班 — 61，夜班 — 62

请分析：
1. 帕累托分布（80/20）— 主要停机原因
2. 与第一季度对比趋势
3. 风险区域（停机增加的设备）
4. 建议（维修、更换、培训）

结果：用于规划的分析报告，节省1.5小时。

13. 维修预算论证

对必要维修支出的财务论证。

提示词示例
请编制3号车间2027年维修预算论证：

原始数据：
- 2026年实际：4,730万₽（计划4,200万，超支12.6%）
- 主要超支项目：压缩机非计划故障（+820万），输送机T-04更换（+410万）
- 2027年计划维修：磨机M-3大修（1,500万），压缩机中修（1,200万），日常维修（1,800万）
- 非计划预留：800万
- 通胀/备件涨价：+8%

总计：5,780万₽（比2026年+22%）

请为财务总监编制论证：为什么+22%，如果削减会有什么后果，有哪些风险。

结果：面向董事会的论证材料，节省2–3小时。

14. 「维修 vs 更换」论证

投资决策 — 修理旧的，还是购买新的。

提示词示例
请论证：压缩机K-05是维修还是更换？

数据：
- 出厂年份：2012（已运行14年）
- 大修预计费用：850万₽
- 新设备同类价格：3,200万₽
- 大修后MTBF（历史数据）：约2,800小时
- 新设备MTBF：约8,000小时
- 旧设备能耗：比新设备高18%
- 年备件费用：210万₽
- 旧设备残值：0

请做对比分析（3年、5年、10年），考虑：
- CAPEX（资本支出）
- OPEX（运营、备件、能耗）
- 停机（生产损失）
- 风险（大修后MTBF不确定性）

结果：基于数据的决策论证，节省3–4小时。

15. 大修承包商报价分析

按价格/工期/经验等标准比较商业报价。

提示词示例
技术任务书：磨机M-3大修（更换衬板、减速器、润滑系统）。

承包商报价：
1. LLC "MetallRemont" — 1,520万，21天，5个类似项目，质保12个月
2. LLC "GorTekhServis" — 1,380万，28天，12个项目，质保18个月
3. LLC "SibRemStroy" — 1,450万，18天，3个项目，质保6个月

评分标准（权重）：
- 价格（30%），工期（25%），经验（20%），质保（15%），口碑（10%）

请制作对比表并给出推荐意见。

结果：有据可依的承包商选择，节省2小时。

知识检验

问题1：为什么大修承包商报价分析属于本地部署？

竞争对手的商业条件、价格、工期 — 属于机密信息
因为技术任务书包含设备的机密图纸
因为备件需要在公开市场寻找

问题2：能否在外部AI中准备泵的维护规程？

可以，如果是通用泵的通用结构，不涉及具体工厂
不可以，任何设备维护文件都包含生产机密
只有隐去工厂名称才可以

问题3：「维修 vs 更换」论证属于哪个环路？

本地部署 — 包含财务数据、生产参数、风险
云端 — TCO计算方法属于公开知识
混合 — 方法论外部，数字内部
