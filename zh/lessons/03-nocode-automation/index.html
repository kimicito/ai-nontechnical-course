3.1 · No-code автоматизация: n8n и Zapier

🎯 Без кода · ✅ Интерактивный квиз

🎯 Зачем это бизнесу

Менеджеры тратят 30% времени на рутину: копирование данных между системами, отправку однотипных писем, заполнение отчётов.

No-code платформы позволяют автоматизировать эти процессы без программирования. Соединяют разные сервисы: email, Excel, CRM, мессенджеры, AI.

Пример: "Когда приходит письмо от поставщика → AI анализирует → если есть изменение цены → отправить уведомление закупщику в Telegram"

📚 Что изучаем

Как работает автоматизация

Любая автоматизация состоит из 3 частей:

  1. Триггер — что запускает процесс? (письмо, время, кнопка)
  2. Действие — что делать? (отправить письмо, записать в таблицу)
  3. Условие — когда делать? (если цена изменилась > 10%)

n8n — open-source автоматизация

🔗 n8n

💰 Бесплатно (self-hosted), облако от $20/мес

n8n — open-source платформа с визуальным редактором workflow. Устанавливается на свой сервер или используется в облаке. Код открыт, лицензия fair-code (бесплатно для себя, платно для облачных SaaS-провайдеров).

Ключевые особенности
  • 400+ интеграций «из коробки» (Google Sheets, Telegram, OpenAI, Notion, CRM)
  • Визуальный редактор — перетаскиваете узлы, соединяете стрелками
  • Self-hosted — данные не покидают ваш сервер (конфиденциальность)
  • HTTP Request — подключение к любому API без программирования
  • JavaScript/Python — можно встроить код для сложной логики
  • Webhook — принимает данные из любых систем
  • Schedule — запуск по расписанию (каждый час, день, крон-выражение)
Пример workflow в n8n
1
Trigger: Webhook (пришёл новый лид с сайта)
2
AI: OpenAI узел — анализирует запрос, определяет категорию
3
Condition: Если категория = "Закупка" → отправить в Telegram закупщику
4
Condition: Если категория = "Продажа" → создать сделку в CRM
5
Action: Записать лог в Google Sheets

Zapier — простота и скорость

⚡ Zapier

💰 Бесплатно до 100 задач/мес, далее от $19/мес

Zapier — самый зрелый инструмент на рынке. 5000+ интеграций, работает с 2011 года. Идеален для быстрого старта без настройки серверов.

Ключевые особенности
  • Zaps — автоматизации, состоящие из триггера и действий
  • Paths — ветвление логики (if/else внутри одного Zap)
  • Filter — условия, при которых Zap срабатывает
  • Formatter — преобразование данных (даты, числа, текст)
  • Schedule — запуск по времени
  • Webhooks — приём данных из внешних систем
  • AI Actions — встроенная интеграция с OpenAI, Anthropic, Cohere
  • Tables — встроенная база данных (альтернатива Google Sheets)
Пример Zap
1
Trigger: Gmail — новое письмо с вложением PDF
2
AI: Zapier AI — извлекает ключевые данные из PDF (сумма, дата, контрагент)
3
Filter: Если сумма > 100 000 ₽
4
Action: Отправить уведомление в Slack → #закупки
5
Action: Сохранить данные в Google Sheets

Когда что выбрать

n8n если:

  • Работаете с конфиденциальными данными (договоры, персональные данные)
  • Есть IT-специалист, который может поднять сервер
  • Нужна сложная логика (циклы, условия, код)
  • Хотите сэкономить на подписке (self-hosted = бесплатна лицензия, но нужен сервер и ИТ-специалист)

Zapier если:

  • Нужно запустить за 10 минут без настройки сервера
  • Используете редкие сервисы (нужна максимальная совместимость)
  • Нет IT-ресурсов — всё должно работать «из коробки»
  • Готовы платить за удобство и стабильность

🛠️ Как применить

Сценарий 1: Автоматическая обработка заявок

Проблема: Заявки от клиентов приходят на email, менеджер вручную переносит их в CRM.

Автоматизация в n8n
1
Email Trigger: Новое письмо на zakazy@company.ru
2
OpenAI: Извлечь: продукт, бюджет, срок, контакт
3
Code: Проверить, все ли поля заполнены
4
CRM: Создать сделку (если поля заполнены)
5
Telegram: Уведомить менеджера
6
If: Бюджет > 100 000 ₽ → уведомить директора

Экономия: 2 часа в день на обработку заявок

Сценарий 2: Мониторинг цен поставщиков

Автоматизация в Zapier
1
Schedule: Еженедельно, например по понедельникам 9:00
2
HTTP: Получить прайс-лист с сайта поставщика (API или парсинг)
3
AI: Сравнить с прошлой неделей (Zapier AI или OpenAI)
4
Filter: Если цена изменилась > 5%
5
Sheets: Записать отчёт в Google Sheets
6
Slack: Отправить уведомление закупщику

Сценарий 3: Еженедельный отчёт

Автоматизация в n8n
1
По расписанию (cron): Каждую пятницу 17:00
2
Database: Собрать данные из 1С / CRM (SQL-запрос или API)
3
Code: Агрегировать данные (группировка, суммы)
4
OpenAI: Сформировать текстовый отчёт с выводами
5
Email: Отправить руководителю
6
Google Drive: Сохранить PDF в архив

👨‍💻 Как делегировать разработчику

Шаблон задачи для n8n

"Нужно создать workflow в n8n.

Триггер: [описать событие]
Действия:
1. [что делать]
2. [что делать]
3. [условие — когда делать]

Интеграции: [список сервисов]
AI-компонент: [да/нет, описание задачи]
Self-hosted: [да/нет]

Пример входных данных: [прикрепить]
Ожидаемый результат: [описать]

Срок: [дата]"

Шаблон задачи для Zapier

"Нужно создать Zap.

Триггер: [сервис + событие]
Действия:
1. [сервис + действие]
2. [сервис + действие]

Условия (Filter/Paths):
- [условие 1]
- [условие 2]

Интеграции: [список сервисов]
AI-компонент: [Zapier AI / OpenAI / нет]

Пример входных данных: [прикрепить]
Ожидаемый результат: [описать]

Срок: [дата]"

✅ Чеклист принятия результата

  • Сценарий запускается корректно? (триггер работает)
  • Данные передаются без потерь? (проверьте 3 запуска)
  • Условия срабатывают правильно? (крайние значения (граничные случаи))
  • Есть обработка ошибок? (если сервис недоступен)
  • Логи доступны для проверки? (история запусков в n8n / Zapier Task History)
  • Стоимость операций в пределах бюджета? (Zapier: счётчик задач, n8n: Execution History)

🎯 Главное из урока

No-code автоматизация — это LEGO для бизнес-процессов. n8n — если нужен контроль данных и сложная логика (open-source, self-hosted). Zapier — если нужен быстрый старт с максимумом интеграций. Триггер → действие → условие. Экономия: 2-4 часа в день на рутине.
