3.1 · Automatisation sans code : n8n et Zapier

🎯 Sans code · ✅ Quiz interactif

🎯 Pourquoi c'est important pour l'entreprise

Les managers dépensent30% du temps en routine: copier des données entre systèmes, envoyer des lettres similaires, remplir des rapports.

Plateformes sans codevous permettent d'automatiser ces processus sans programmation. Ils connectent différents services : email, Excel, CRM, messageries instantanées, IA.

Exemple:"Quand une lettre arrive d'un fournisseur → l'IA analyse → s'il y a un changement de prix → envoyer une notification à l'acheteur dans Telegram"

📚 Ce que nous étudions

Comment fonctionne l'automatisation

Toute automatisation se compose de 3 parties :

  1. Déclenchement— qu'est-ce qui déclenche le processus ? (lettre, heure, bouton)
  2. Action- ce qu'il faut faire? (envoyer une lettre, l'écrire dans un tableau)
  3. Condition— quand le faire ? (si le prix a changé > 10%)

n8n - automatisation open source

🔗 n8n

💰 Gratuit (auto-hébergé), cloud à partir de 20$/mois

n8n — open-sourceplateforme avec un éditeur visuel de workflow. Installé sur votre serveur ou utilisé dans le cloud. Code ouvert, licencefair-code(gratuit pour vous-même, payant pour les fournisseurs cloud SaaS).

Principales fonctionnalités
  • Plus de 400 intégrations« prêt à l'emploi » (Google Sheets, Telegram, OpenAI, Notion, CRM)
  • Éditeur visuel— faites glisser les nœuds, connectez-les avec des flèches
  • Self-hosted— les données ne quittent pas votre serveur (confidentialité)
  • HTTP Request— connexion à n'importe quelle API sans programmation
  • JavaScript/Python- vous pouvez intégrer du code pour une logique complexe
  • Webhook— accepte les données de n'importe quel système
  • Schedule— lancement programmé (chaque heure, jour, expression de la couronne)
Exemple de flux de travail dans n8n
1
Trigger:Webhook (un nouveau lead est arrivé du site)
2
AI:Nœud OpenAI - analyse la demande, détermine la catégorie
3
Condition:Si catégorie = "Achat" → envoyer à l'acheteur dans Telegram
4
Condition:Si catégorie = "Vente" → créer une transaction dans CRM
5
Action:Écrire un journal sur Google Sheets

Zapier - simplicité et rapidité

⚡ Zapier

💰 Gratuit jusqu'à 100 tâches/mois, puis à partir de 19$/mois

Zapier est l'outil le plus mature du marché.Plus de 5 000 intégrations, est opérationnel depuis 2011. Idéal pour un démarrage rapide sans configuration de serveurs.

Principales fonctionnalités
  • Zaps— des automatisations composées d'un déclencheur et d'actions
  • Paths— logique de branchement (if/else dans un Zap)
  • Filter— conditions dans lesquelles Zap est déclenché
  • Formatter— conversion de données (dates, chiffres, texte)
  • Schedule- commencer par l'heure
  • Webhooks— recevoir des données de systèmes externes
  • AI Actions- intégration intégrée avec OpenAI, Anthropic, Cohere
  • Tables— base de données intégrée (alternative à Google Sheets)
Exemple de Zap
1
Trigger:Gmail - nouvel e-mail avec pièce jointe PDF
2
AI:Zapier AI - extrait les données clés du PDF (montant, date, contrepartie)
3
Filter:Si le montant est > 100 000 ₽
4
Action:Envoyer une notification à la messagerie d'entreprise → #achats
5
Action:Enregistrer les données dans Google Sheets

Quand choisir quoi

n8nSi:

  • Vous travaillez avec des données confidentielles (contrats, données personnelles)
  • Il y a un informaticien qui peut installer le serveur
  • Nécessite une logique complexe (boucles, conditions, code)
  • Vous souhaitez économiser sur un abonnement (auto-hébergé = licence gratuite, mais nécessite un serveur et un informaticien)

ZapierSi:

  • Doit être lancé en 10 minutes sans configuration du serveur
  • Vous utilisez des services rares (vous avez besoin d'une compatibilité maximale)
  • Aucune ressource informatique : tout devrait fonctionner immédiatement
  • Êtes-vous prêt à payer pour la commodité et la stabilité ?

🛠️ Comment postuler

Scénario 1 : Traitement automatisé des commandes

Problème:Les candidatures des clients arrivent par email et le manager les transfère manuellement vers le CRM.

Automatisation dans n8n
1
Email Trigger:Nouvelle lettre à zakazy@company.ru
2
OpenAI:Extrait : produit, budget, délai, contact
3
Code:Vérifiez si tous les champs sont remplis
4
CRM:Créer une transaction (si les champs sont renseignés)
5
Telegram:Informer le gestionnaire
6
If:Budget > 100 000 ₽ → prévenir le directeur

Économie:2 heures par jour pour traiter les candidatures

Scénario 2 : Suivi des prix des fournisseurs

Automatisation à Zapier
1
Schedule:Hebdomadaire, par exemple le lundi à 9h00
2
HTTP:Obtenir une liste de prix sur le site Web du fournisseur (API ou parsing)
3
AI:Comparer avec la semaine dernière (Zapier AI ou OpenAI)
4
Filter:Si le prix a changé > 5%
5
Sheets:Rédiger un rapport dans Google Sheets
6
Messagerie d'entreprise :Envoyer une notification à l'acheteur

Scénario 3 : rapport hebdomadaire

Automatisation dans n8n
1
Par planning (cron) :Tous les vendredis à 17h00
2
Database:Collecter des données depuis 1C / CRM (requête SQL ou API)
3
Code:Données agrégées (regroupement, sommes)
4
OpenAI:Générer un rapport texte avec les résultats
5
Email:Envoyer au gestionnaire
6
Google Drive:Enregistrer le PDF pour archiver

Scénario 4 : Rapport matinal sur les arrêts d'équipement

Automatisation pour la production
1
Par planning (cron) : Chaque jour à 07:30
2
Database / SCADA : Extraire le journal des arrêts pour la journée écoulée
3
Code : Regrouper par ateliers et équipements, calculer le temps total
4
GENAI (on-prem) : Analyser les causes, identifier les pannes récurrentes, donner des recommandations sur les pièces de rechange
5
Messagerie d'entreprise : Envoyer le résumé au chef d'atelier et à l'ingénieur en chef
6
Sheets : Ajouter une ligne dans le tableau pour accumuler les statistiques

✅ Checklist d'acceptation

  • Le script s'exécute-t-il correctement ? (le déclencheur fonctionne)
  • Les données sont-elles transférées sans perte ? (vérifiez 3 courses)
  • Les conditions fonctionnent-elles correctement ? (valeurs extrêmes (cas limites))
  • Y a-t-il une gestion des erreurs ? (si le service n'est pas disponible)
  • Les journaux sont-ils disponibles pour examen ? (exécuter l'historique dans n8n / Zapier Task History)
  • Coût des opérations dans les limites du budget ? (Zapier : compteur de tâches, n8n : Historique d'exécution)
