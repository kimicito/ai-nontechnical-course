1.2 · Sortie structurée : JSON, tableaux, listes de contrôle

🎯 Sans code · ✅ Quiz interactif

🎯 Pourquoi c'est important pour l'entreprise

L'IA peut apporter des réponsessous n'importe quel format. Mais la valeur par défaut est un texte solide que vous devez interrompre manuellement dans Excel ou 1C.

Problème:Le manager reçoit l'analyse sous forme de texte et passe 15 minutes à la transférer sur la table.

Solution:Demandez à l'IA de produire immédiatement un tableau. Économie:15 minutes × 20 tâches = 5 heures par semaine.

📚 Ce que nous étudions

4 formats de sortie

📊 Tableau

Comparaison, analyse, rapports. Facile à copier vers Excel.

📋 JSON

Pour les programmeurs et l'automatisation. Données structurées.

☑️ Liste de contrôle

Pour inspection, audit, contrôle qualité.

📝 Markdown

Pour la documentation, les instructions, les articles. Lisible partout.

Comment demander une table

💡 Exemple rapide
Vous êtes analyste financier.
Analyser 3 projets d'investissement (données ci-dessous).

Affichez le résultat sous forme de tableau :
| Projet | Retour sur investissement | Période de récupération | Risque | Recommandation |

Le tableau suit une brève justification de la recommandation.

Comment demander une liste de contrôle

💡 Exemple rapide
Vous êtes auditeur ISO 9001.
Effectuer un audit du processus d’approvisionnement en utilisant les critères suivants.

Affichez le résultat sous forme de liste de contrôle :
☐ Critère 1 - [statut : réussi/échec] - [commentaire]
☐ Critère 2 - [statut] - [commentaire]

A la fin : score global de conformité (en pourcentage).

🛠️ Comment postuler

Scénario 1 : Comparaison des fournisseurs

Tâche:Sélectionnez un fournisseur d'emballage parmi 3 options.

Rapide
Vous êtes un acheteur FMCG.
Comparez 3 phrases selon les critères et affichez-les dans un tableau :
| Critère | Fournisseur A | Fournisseur B | Fournisseur B | Chef |
| Prix ​​par unité | | | | |
| Min. commander | | | | |
| Délai de livraison | | | | |
| Conditions de paiement | | | | |

Donnez une recommandation : qui choisir et pourquoi.

Résultat:Table prête à être présentée à la direction. Copié sur Excel en un seul clic.

Scénario 2 : Audit du contrat

Tâche:Vérifiez le contrat pour les risques.

Rapide
Vous êtes avocat d'entreprise.
Analyser le contrat de fourniture pour les risques.

Affichez le résultat à l'aide d'une liste de contrôle :
☐ Risque : [description] - Niveau : [Élevé/Moyen/Faible] - Recommandation : [que faire]

A la fin : une conclusion générale sur l'acceptabilité du contrat.

Scénario 3 : rapport hebdomadaire

Tâche:Préparer un rapport de ventes hebdomadaire.

Rapide
Vous êtes analyste commercial.
Préparez un rapport hebdomadaire basé sur les données ci-dessous.

Format :
1. Indicateurs clés (tableau : Indicateur | Valeur | Δ à la semaine dernière)
2. Top 3 des tendances (liste)
3. Recommandations (liste avec priorités)
4. Risques (tableau : Risque | Probabilité | Impact)

👨‍💻 Comment déléguer à un développeur

Modèle de tâche

« Nous avons besoin d’un analyseur de réponses IA dans des formats standardisés.

Exigences :
1. Reconnaît automatiquement le format (table/JSON/liste de contrôle)
2. Convertit les tableaux en Excel (.xlsx)
3. Convertit JSON en CSV
4. Vérifie l'intégrité des données (pas de champs vides)
5. Point de terminaison de l'API pour l'intégration

Exemple de données d'entrée (joindre) :
[insérer un exemple de réponse de l'IA]

Résultat attendu (joindre) :
[insérer Excel/CSV souhaité]

Date limite : [date]"

✅ Checklist d'acceptation

  • Le format correspond-il à la demande ? (tableau/JSON/liste de contrôle)
  • Tous les champs sont-ils remplis ? (pas de laissez-passer)
  • Les données sont-elles correctes ? (cochez 2-3 cellules aléatoires)
  • Puis-je le copier sur Excel / 1C / CRM ?
  • La structure est-elle claire sans explication ?
  • Les unités de mesure sont-elles précisées ? (roubles, jours, intérêts)
