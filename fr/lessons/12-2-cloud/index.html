Niveau 13 — Logistique

13.2 Tâches logistiques externes

⏱ 15 min · pour les logisticiens et les spécialistes de la chaîne d'approvisionnement
Données ouvertes (IA externe)
Toutes les tâches de cette leçon utilisent des sources publiques, des méthodologies générales, le marché ouvert. Vous pouvez utiliser Claude, ChatGPT, Gemini.

Pourquoi le cloud est meilleur que l'on-prem pour la logistique

Les modèles cloud l'emportent quand il faut :

  • Recherche web : surveiller les indices de fret, les tarifs des chemins de fer russes, les restrictions publiques
  • Méthodologie : élaborer une stratégie générale sans volumes réels
  • Benchmarking : comparer les pratiques industrielles 3PL, KPI
  • Prévisions : résumer les prévisions publiques météo/géopolitiques
  • Modèles : créer des documents types (CMR, demandes)
Important : avant d'envoyer dans le cloud — vérifiez, qu'il n'y a pas de volumes réels, d'itinéraires, de tarifs, de noms de contreparties dans la requête. Si c'est le cas — anonymisez ou envoyez en on-prem.

1. Élaboration d'une stratégie logistique générale

Méthodologie de construction d'un réseau d'itinéraires — sans lien avec une entreprise spécifique.

Exemple de prompt (Claude avec recherche web)
Préparez la structure d'une stratégie logistique pour une entreprise métallurgique :

Sections :
1. Analyse du réseau actuel (étapes typiques : matières premières → production → client)
2. Choix du mode de transport (mer/chemin de fer/route/conteneur maritime) — critères
3. Réservation et redondance (itinéraires de secours)
4. Gestion des stocks (stock de sécurité, point de commande)
5. 3PL vs interne — critères de choix
6. Systèmes IT (TMS, WMS, EDI)
7. KPI (OTIF, coût par tonne, délai de livraison, empreinte carbone)
8. Gestion des risques (planification de scénarios, assurance)

Sans volumes ni itinéraires spécifiques.

Résultat : stratégie prête, économise 5–6 heures.

2. Surveillance des indices de fret

Baltic Dry Index et analogues — données publiques pour le benchmarking.

Exemple de prompt (recherche web)
Effectuez une surveillance des indices de fret (juillet 2026) :

Indices :
1. Baltic Dry Index (BDI) — dynamique sur 3 mois
2. Baltic Capesize Index (BCI)
3. Baltic Panamax Index (BPI)
4. China Containerized Freight Index (CCFI)

Questions :
- Quelle est la dynamique depuis le début de l'année ?
- Quels facteurs influencent (météo, géopolitique, saisonnalité) ?
- Prévisions pour T3–T4 2026 (selon les analystes)
- Quels itinéraires augmentent le plus vite ?
- Recommandations pour le couverture des risques de fret

Présentez sous forme de tableau avec les sources.

Résultat : aperçu du marché du fret, économise 2–3 heures.

3. Analyse des tarifs des chemins de fer russes et des opérateurs chinois

Grilles tarifaires publiques — comparaison, recherche d'itinéraires optimaux.

Exemple de prompt (recherche web)
Comparez les tarifs du transport ferroviaire de conteneurs :

Itinéraire (type) : [INDIQUEZ LA GARE] (RF) → [INDIQUEZ LA GARE] (RPC)

Opérateurs :
1. Chemins de fer russes (transport de conteneurs) — tarif pour 40 pieds
2. Opérateurs chinois (CR, China Railway)
3. Transit via le Kazakhstan (KTZ)
4. Opérateurs multimodaux (FESCO, TransContainer)

En complément :
- Délais de livraison
- Fréquence des départs
- Disponibilité des wagons/plateformes
- Frais supplémentaires (passage frontalier, terminaux)

Présentez un tableau comparatif.

Résultat : comparaison des tarifs, économise 2 heures.

4. Analyse des goulots d'étranglement aux passages frontaliers

Situation générale selon les sources ouvertes — non liée à une livraison spécifique.

Exemple de prompt (recherche web)
Analysez la situation aux passages frontaliers RF–RPC (2026) :

Passages :
1. [INDIQUEZ LE PASSAGE 1]
2. [INDIQUEZ LE PASSAGE 2]
3. [INDIQUEZ LE PASSAGE 3]
4. [INDIQUEZ LE PASSAGE 4]

Données à collecter :
- Temps d'attente moyen (jours)
- Capacité de passage (wagons/jour)
- Problèmes typiques (documents, files d'attente, météo)
- Infrastructure (entrepôts, terminaux)
- Plans de développement (investissements, extension)

Sources : statistiques officielles des douanes, médias spécialisés, forums de transporteurs.

Résultat : aperçu des goulots d'étranglement, économise 3–4 heures.

5. Collecte et résumé des prévisions météo/catastrophes

Fonction générative — résumé des bulletins publics, pas une prévision en soi.

Exemple de prompt
Résumez les prévisions ouvertes influençant les itinéraires de livraison en août 2026 :

Régions :
- Mer de Chine méridionale (typhons)
- Cap de Bonne-Espérance (tempêtes)
- Canal de Suez (météo, géopolitique)
- Mer Baltique (tempêtes d'automne)

Pour chacune :
1. Prévisions publiques (services météorologiques, NOAA)
2. Période de risque (dates)
3. Probabilité et intensité
4. Impact sur la navigation (fermeture de ports, restrictions)
5. Recommandations (éviter, contourner, réserver)

Format : alerte pour le service logistique.

Résultat : synthèse des risques, économise 1,5 heure.

6. Benchmarking des coûts et pratiques logistiques

Comparaison avec les standards de l'industrie — données publiques.

Exemple de prompt (recherche web)
Effectuez un benchmarking des coûts logistiques dans la métallurgie (2024–2026) :

Questions :
1. Quel % du chiffre d'affaires représentent les dépenses logistiques chez les grands métallurgistes ?
2. Quels KPI utilisent les leaders de l'industrie (OTIF, coût par tonne, intensité carbone) ?
3. Quels prestataires 3PL travaillent avec la métallurgie (FESCO, TransContainer, DP World) ?
4. Quels systèmes TMS sont déployés (SAP TM, 1C Logistique, Ortec) ?
5. Quelles tendances : green logistics, jumeau numérique, blockchain dans la documentation ?

Rédigez un rapport structuré avec les sources.

Résultat : aperçu des pratiques industrielles, économise 4–5 heures.

7. Modèle général de documents pour le transport international routier

CMR, autorisations types — sans marchandises réelles.

Exemple de prompt
Préparez un modèle de dossier de documents pour le transport international routier (RF → UE) :

Documents :
1. CMR (connaissement routier) — champs obligatoires
2. Lettre de voiture (document de transport)
3. Facture (facture commerciale)
4. Liste de colisage
5. Certificat d'origine (Form A, EUR.1)
6. Permis de transport (si marchandises dangereuses — ADR)
7. Police d'assurance (assurance CMR)
8. Procuration pour la réception de la marchandise

Pour chacun : objet, champs obligatoires, erreurs typiques.
Emplacement pour l'insertion de données réelles : [À PRÉCISER].

Résultat : modèle prêt, économise 2–3 heures.

8. Préparation d'alertes pour détournement/report d'expédition

Scénario général de routage — sans lien avec une marchandise spécifique.

Exemple de prompt
Préparez une méthodologie d'alertes et de recommandations en cas de perturbations logistiques :

Scénarios types :
1. Tempête dans le port de chargement — retard de 3–5 jours
2. Surcharge au passage frontalier — file d'attente 10+ jours
3. Grève dans le port de déchargement — retard indéterminé
4. Accident sur les voies ferrées — détour par un itinéraire alternatif
5. Restrictions réglementaires — interdiction soudaine d'accès au port

Pour chaque scénario :
- Déclencheur de l'alerte (ce que nous surveillons)
- Qui est notifié (logisticien, acheteur, client)
- Options d'action (détour, report, remplacement)
- Modèle de message au partenaire
- Délai de décision

Méthodologie sans marchandises ni dates spécifiques.

Résultat : méthodologie prête, économise 3–4 heures.

Vérification des connaissances

Question 1 : Pourquoi la stratégie logistique — en IA externe ?

Parce que c'est une méthodologie générale sans volumes ni itinéraires spécifiques
Parce que la stratégie n'est pas un document, mais un concept
Seulement si l'on anonymise le nom de l'entreprise

Question 2 : Baltic Dry Index — quel contour ?

IA externe — indice de fret public
On-prem — secret commercial
Hybride — indice à l'extérieur, interprétation à l'intérieur

Question 3 : Modèle CMR — quel contour ?

IA externe — document type sans marchandises réelles
On-prem — CMR contient des données commerciales
Hybride — modèle à l'extérieur, données à l'intérieur
