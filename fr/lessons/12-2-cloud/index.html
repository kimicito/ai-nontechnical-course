13.2 Tâches logistiques externes

⏱ 15 мин · для логистов и специалистов по цепочкам поставок
Données publiques (IA externe)
Все задачи этого урока — с публичными источниками, общими методологиями, открытым рынком. Можно использовать Claude, ChatGPT, Gemini.

Почему облако лучше on-prem для логистики

Облачные модели выигрывают, когда нужно:

  • Веб-поиск: мониторить фрахтовые индексы, тарифы РЖД, публичные ограничения
  • Методология: разработать общую стратегию без реальных объёмов
  • Бенчмаркинг: сравнить отраслевые практики 3PL, KPI
  • Прогнозы: суммаризировать публичные прогнозы погоды/геополитики
  • Шаблоны: создать типовые документы (CMR, заявки)
Important: перед отправкой в облако — проверь, что в запросе нет реальных объёмов, маршрутов, ставок, названий контрагентов. Если есть — обезличь или отправь в on-prem.

1. Разработка логистической стратегии в общем виде

Методология построения сети маршрутов — без привязки к конкретной компании.

Пример промпта (Claude с веб-поиском)
Подготовь структуру логистической стратегии для металлургической компании:

Разделы:
1. Анализ текущей сети (типичные этапы: сырьё → производство → клиент)
2. Выбор режима транспорта (море/ЖД/авто/морской контейнер) — критерии
3. Резервирование и redundance (дублирование маршрутов)
4. Управление запасами (safety stock, reorder point)
5. 3PL vs in-house — критерии выбора
6. IT-системы (TMS, WMS, EDI)
7. KPI (OTIF, cost per ton, lead time, carbon footprint)
8. Риск-менеджмент (сценарное планирование, insurance)

Без конкретных объёмов и маршрутов.

Результат: готовая стратегия, экономит 5–6 часов.

2. Мониторинг фрахтовых индексов

Baltic Dry Index и аналоги — публичные данные для бенчмаркинга.

Пример промпта (веб-поиск)
Проведи мониторинг фрахтовых индексов (июль 2026):

Индексы:
1. Baltic Dry Index (BDI) — динамика за 3 месяца
2. Baltic Capesize Index (BCI)
3. Baltic Panamax Index (BPI)
4. China Containerized Freight Index (CCFI)

Questionы:
- Какая динамика с начала года?
- Какие факторы влияют (погода, геополитика, сезонность)?
- Прогноз на Q3–Q4 2026 (по аналитикам)
- Какие маршруты дорожают быстрее всего?
- Рекомендации по хеджированию фрахтовых рисков

Выведи таблицу с источниками.

Результат: обзор рынка фрахта, экономит 2–3 часа.

4. Анализ узких мест на погранпереходах

Общая ситуация по открытым источникам — не привязанная к конкретной поставке.

Пример промпта (веб-поиск)
Проанализируй ситуацию на погранпереходах РФ–КНР (2026):

Переходы:
1. [УКАЖИТЕ ПЕРЕХОД 1]
2. [УКАЖИТЕ ПЕРЕХОД 2]
3. [УКАЖИТЕ ПЕРЕХОД 3]
4. [УКАЖИТЕ ПЕРЕХОД 4]

Данные для сбора:
- Среднее время ожидания (дни)
- Пропускная способность (вагонов/сутки)
- Типичные проблемы (документы, очереди, погода)
- Инфраструктура (склады, терминалы)

Источники: официальная статистика таможни, отраслевые СМИ, форумы перевозчиков.

Результат: обзор узких мест, экономит 3–4 часа.

5. Сбор и суммаризация прогнозов погоды/катаклизмов

Генеративная функция — суммаризация публичных сводок, не прогноз как таковой.

Пример промпта
Суммаризируй открытые прогнозы, влияющие на маршруты доставки в августе 2026:

Регионы:
- Южно-Китайское море (тайфуны)
- Мыс Доброй Надежды (штормы)
- Суэцкий канал (погода, геополитика)
- Балтийское море (осенние шторма)

Для каждого:
1. Публичные прогнозы (метеорологические службы, NOAA)
2. Период риска (даты)
3. Вероятность и сила
4. Влияние на судоходство (закрытие портов, ограничения)
5. Рекомендации (избегать, обходить, резервировать)

Формат: алерт для логистического отдела.

Результат: сводка рисков, экономит 1.5 часа.

Пример промпта (веб-поиск)

Сделай структурированный отчёт с источниками.

Результат: обзор отраслевых практик, экономит 4–5 часов.

7. Общий шаблон документов для международной автоперевозки

CMR, типовые разрешения — без реальных грузов.

Пример промпта
Подготовь шаблон пакета документов для международной автоперевозки (РФ → ЕС):

Документы:
1. CMR (коносамент для авто) — обязательные поля
2. ТТН (товарно-транспортная накладная)
3. Инвойс (коммерческий счёт)

7. Страховой полис (CMR-страхование)
8. Доверенность на получение груза

Для каждого: назначение, обязательные поля, типичные ошибки.
Место для вставки реальных данных: [УТОЧНИТЬ].

Результат: готовый шаблон, экономит 2–3 часа.

8. Подготовка алертов по объезду/переносу отгрузки

Общий сценарий маршрутизации — без привязки к конкретному грузу.

Пример промпта
Подготовь методологию алертов и рекомендаций при сбоях в логистике:

Типовые сценарии:
1. Шторм в порту погрузки — задержка на 3–5 дней
2. Перегруз погранперехода — очередь 10+ дней
3. Забастовка в порту выгрузки — неопределённая задержка
4. Авария на ЖД путях — обход через альтернативный маршрут
5. Регуляторные ограничения — внезапный запрет на заход в порт

Для каждого сценария:
- Триггер алерта (что фиксируем)
- Кто уведомляется (логист, закупщик, клиент)
- Варианты действий (обход, перенос, замена)
- Шаблон сообщения контрагенту
- Срок принятия решения

Методология без конкретных грузов и дат.

Результат: готовая методология, экономит 3–4 часа.

Test de connaissances

Question 1: Почему стратегия логистики — во внешний ИИ?

Потому что это общая методология без конкретных объёмов и маршрутов
Потому что стратегия — это не документ, а концепция
Только если обезличить название компании

Question 2: Baltic Dry Index — какой контур?

Внешний ИИ — публичный фрахтовый индекс
On-prem — коммерческая тайна
Гибрид — индекс снаружи, интерпретация внутри

Question 3: Шаблон CMR — какой контур?

Внешний ИИ — типовой документ без реальных грузов
On-prem — CMR содержит коммерческие данные
Гибрид — шаблон снаружи, данные внутри
