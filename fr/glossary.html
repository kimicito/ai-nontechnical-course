📖 Glossaire

Concepts complexes de l'IA expliqués en mots simples. Si vous rencontrez un mot inconnu dans le cours — cherchez-le ici.

🔤 Termes clés

API

« Prise de données » — un moyen pour les programmes de se connecter entre eux

Si le site A veut obtenir des données du programme B, il utilise une API. Par exemple, quand une calculatrice de cours obtient le taux actuel du dollar sur le site de la Banque centrale.

Open-source

Logiciel avec code source ouvert — peut être téléchargé et utilisé gratuitement

Les développeurs ont publié le code publiquement. N'importe qui peut le télécharger, l'étudier et le modifier. Exemples : n8n, LibreOffice. Mais : le support et la formation peuvent être payants (comme Red Hat).

Self-hosted

« Sur votre propre serveur » — le programme fonctionne à l'intérieur de l'entreprise, pas dans le cloud

Les données ne quittent jamais le bureau. Nécessite un serveur et un spécialiste IT pour la configuration. La licence est gratuite, mais le matériel et la maintenance ne le sont pas.

VPN

Réseau privé virtuel — un tunnel sécurisé pour le travail à distance

Permet à un employé travaillant de chez lui de se connecter en toute sécurité au réseau d'entreprise. Pour l'IA on-premise au bureau, un réseau local est nécessaire ; pour le travail à distance — un VPN.

Firewall

Barrière réseau — protection contre les connexions non désirées

Comme un gardien à l'entrée : ne laisse passer que ceux qui ont l'autorisation. Configuré par le département IT.

Cron / Planning

Exécution d'une tâche selon un planning : chaque jour, chaque heure, etc.

Par exemple : « Chaque lundi à 9h00 vérifier les prix des fournisseurs. » Dans n8n cela s'appelle Schedule, en programmation — cron.

JSON

Format de données pour les programmes — « un tableau sous forme de texte »

L'IA fournit souvent des réponses au format JSON, qui est ensuite converti en Excel/tableau. Pour les humains, cela ressemble à du texte avec des accolades.

CSV

Format tabulaire — s'ouvre dans Excel

Format texte simple où les données sont séparées par des virgules. N'importe quel tableau peut être enregistré au format CSV.

LLM (Large Language Model)

Grand modèle de langage — le « cerveau » de l'IA qui comprend et génère du texte

ChatGPT, Claude, Kimi — tous sont des LLM. Entraînés sur une quantité énorme de textes pour comprendre les questions et donner des réponses pertinentes.

Token

Unité de mesure de texte pour l'IA — environ les ¾ d'un mot

L'IA compte non pas les mots, mais les tokens. 1000 tokens ≈ 750 mots. Le coût d'une requête dépend du nombre de tokens.

RAG (Retrieval-Augmented Generation)

Recherche + génération — l'IA trouve d'abord les faits, puis répond

Au lieu de se fier uniquement à la mémoire, l'IA recherche des informations dans vos documents et donne une réponse sur leur base. Réduit le risque d'inventions.

Base de données vectorielle

Stockage où l'IA recherche par sens, et non par mots exacts

Si vous recherchez « fournisseur cher », il trouvera aussi « prix élevé » et « pas bon marché ». Il comprend le contexte, pas seulement la formulation.

Prompt

Une requête ou instruction pour l'IA — ce que vous écrivez dans le chat

Plus le prompt est précis, meilleure est la réponse. Un bon prompt contient : le rôle, la tâche, le format, les contraintes, des exemples.

Agent IA

Un programme qui effectue des tâches de manière autonome selon des instructions données

Il ne se contente pas de répondre aux questions, il agit : recherche, compare, envoie des emails, remplit des documents.

Hallucination

Quand l'IA fournit avec assurance des informations inventées

Peut inventer une loi inexistante, une fausse citation ou un fait erroné. Vérifiez toujours les données critiques.

Fine-tuning

Entraînement d'un modèle sur vos données pour des tâches spécifiques

Un modèle prêt à l'emploi est « ajusté » à vos documents, terminologie et style. Plus cher que l'utilisation normale, mais plus précis.

Cas limites (Edge cases)

Situations extrêmes, non standard — quand tout ne se passe pas comme prévu

Par exemple : budget = 0, fichier vide, fournisseur ne répond pas. Une bonne automatisation prend en compte aussi ces scénarios.

MCP (Model Context Protocol)

Standard de connexion de l'IA aux systèmes externes

Permet à un agent IA d'accéder à vos bases de données, CRM et emails. Comme un connecteur universel pour différents appareils.

On-premise

« Sur site » — tout fonctionne à l'intérieur de l'entreprise, pas dans le cloud

Les données restent dans vos murs. Sécurisé, mais plus cher à maintenir. Adapté aux informations confidentielles.

Workflow

Une séquence d'étapes d'automatisation

Par exemple : recevoir un email → extraire les données → écrire dans un tableau → envoyer une notification. Dans n8n et Zapier, les workflows se créent visuellement.

Webhook

« Signal » d'un programme à un autre — notification instantanée

Quand un événement se produit (par exemple, une nouvelle commande), le système A informe instantanément le système B. Sans interrogation constante.

⚙️ Paramètres, automatisation et audit

Fenêtre de contexte

« Mémoire du modèle » — combien de texte l'IA retient à la fois

Si le document est plus long que la fenêtre, le modèle oublie le début. Pour analyser un contrat de 50 pages, il faut un LLM avec une grande fenêtre (200K+ tokens) ou une technique de chunking.

Température

« Niveau de créativité » de l'IA — du strict à l'imaginatif

0,1 — prévisible, presque déterministe (pour les contrats, rapports). 0,7 — créatif, génère des variantes (pour le brainstorming, marketing). 1,0+ — peut commencer à inventer.

Embedding

« Empreinte numérique du sens » — comment l'IA comprend que deux textes sont similaires

Chaque morceau de texte se voit attribuer un ensemble de nombres (vecteur). L'IA trouve les documents pertinents par « proximité mathématique », même si les mots sont complètement différents.

Chunking

« Découpage en morceaux » — division d'un long document en parties

Un contrat de 100 pages est découpé en paragraphes de 500 tokens chacun, pour que chaque morceau tienne dans la mémoire du modèle. Ensuite l'IA trouve les morceaux nécessaires et assemble la réponse.

Parsing

« Extraction de structure du chaos » — transformation de données non structurées en tableau

Extraire prix, références, unités de mesure d'un catalogue PDF fournisseur vers Excel. Ou extraire dates, montants et adresses d'une correspondance email.

SaaS

Logiciel par abonnement via Internet — pas d'installation, payez et utilisez

ChatGPT, n8n Cloud, Zapier — tous sont du SaaS. Accès via navigateur, données sur les serveurs du fournisseur. Pratique, mais pour les documents confidentiels il faut une alternative on-premise.

Scoring

Évaluation par points — sélection objective des fournisseurs ou contrôle qualité

Un fournisseur obtient 85/100 — passe, 60/100 — éliminé. Critères : prix, délais, réputation, certificats, expérience. L'IA peut calculer automatiquement selon une échelle donnée.

Conformité (Compliance)

Conformité aux règles — lois, normes, règlements internes

Vérification qu'un contrat ne viole pas les lois, qu'un devis est conforme aux normes, qu'un achat a passé toutes les validations. L'IA croise le document avec une liste de contrôle des exigences.

SLA

Accord sur les niveaux de service — temps de réponse, disponibilité, pénalités

« Réponse à la demande — sous 2 heures, disponibilité 99,9 %, pénalité d'indisponibilité — 1 % du tarif mensuel. » L'IA peut surveiller le respect du SLA et signaler les violations.

Pipeline

« Convoyeur » — une chaîne d'étapes de traitement des données

Étape 1 : recevoir un email → Étape 2 : extraire les données → Étape 3 : écrire dans le CRM → Étape 4 : envoyer une notification. Dans n8n et Zapier, les pipelines se construisent visuellement, sans programmation.

🏷️ Abréviations du cours

GENAi

Service interne de l'entreprise basé sur OpenWebUI

Fonctionne sur vos serveurs (on-premise). Sécurisé pour les documents confidentiels. Utilisez GENAi pour : contrats, rapports, données personnelles, protocoles internes.

NDA

Accord de confidentialité — contrat de non-divulgation

Signé avec les fournisseurs d'IA pour qu'ils n'utilisent pas vos données pour entraîner leurs modèles.

ROI

Retour sur investissement — rentabilité des investissements

Montre combien d'argent rapporte l'implémentation de l'IA par rapport aux coûts. ROI = (Profit − Coûts) / Coûts × 100 %.

CRM

Système de gestion de la relation client

Stocke les informations sur les clients, les transactions, les contacts. Exemples : amoCRM, Bitrix24, Salesforce.

ERP

Système de planification des ressources de l'entreprise

Unifie la comptabilité, la production, les achats et les ventes. Exemples : 1C, SAP, Oracle.
