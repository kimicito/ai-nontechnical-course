1.2 · Структурированный вывод: JSON, таблицы, чеклисты

🎯 Без кода · ✅ Интерактивный квиз

🎯 Зачем это бизнесу

AI может выдавать ответы в любом формате. Но по умолчанию — сплошной текст, который приходится перебивать руками в Excel или 1С.

Проблема: Менеджер получает анализ в виде текста, тратит 15 минут на перенос в таблицу.

Решение: Попросить AI сразу выдать таблицу. Экономия: 15 минут × 20 задач = 5 часов в неделю.

📚 Что изучаем

4 формата вывода

📊 Таблица

Сравнение, анализ, отчёты. Легко копировать в Excel.

📋 JSON

Для программистов и автоматизации. Структурированные данные.

☑️ Чеклист

Для проверки, аудита, контроля качества.

📝 Markdown

Для документации, инструкций, статей. Читается везде.

Как попросить таблицу

💡 Пример промпта
Ты — финансовый аналитик.
Проанализируй 3 инвестиционных проекта (данные ниже).

Выведи результат в формате таблицы:
| Проект | ROI | Срок окупаемости | Риск | Рекомендация |

После таблицы — краткое обоснование рекомендации.

Как попросить чеклист

💡 Пример промпта
Ты — аудитор по ISO 9001.
Проведи аудит процесса закупок по следующим критериям.

Выведи результат в виде чеклиста:
☐ Критерий 1 — [статус: пройден/не пройден] — [комментарий]
☐ Критерий 2 — [статус] — [комментарий]

В конце: общая оценка соответствия (в процентах).

🛠️ Как применить

Сценарий 1: Сравнение поставщиков

Задача: Выбрать поставщика упаковки из 3 вариантов.

Промпт
Ты — закупщик FMCG.
Сравни 3 предложения по критериям и выведи таблицей:
| Критерий | Поставщик А | Поставщик Б | Поставщик В | Лидер |
| Цена за ед. | | | | |
| Мин. заказ | | | | |
| Срок доставки | | | | |
| Условия оплаты | | | | |

Дай рекомендацию: кого выбрать и почему.

Результат: Готовая таблица для презентации руководству. Копируется в Excel одним кликом.

Сценарий 2: Аудит контракта

Задача: Проверить договор на риски.

Промпт
Ты — корпоративный юрист.
Проанализируй договор поставки на риски.

Выведи результат чеклистом:
☐ Риск: [описание] — Уровень: [Высокий/Средний/Низкий] — Рекомендация: [что сделать]

В конце: общий вывод о приемлемости договора.

Сценарий 3: Еженедельный отчёт

Задача: Подготовить отчёт о продажах за неделю.

Промпт
Ты — бизнес-аналитик.
На основе данных ниже подготовь еженедельный отчёт.

Формат:
1. Ключевые показатели (таблица: Показатель | Значение | Δ к прошлой неделе)
2. Топ-3 тренда (список)
3. Рекомендации (список с приоритетами)
4. Риски (таблица: Риск | Вероятность | Влияние)

👨‍💻 Как делегировать разработчику

Шаблон задачи

"Нужен парсер ответов AI в стандартизированные форматы.

Требования:
1. Автоматически распознаёт формат (таблица/JSON/чеклист)
2. Конвертирует таблицы в Excel (.xlsx)
3. Конвертирует JSON в CSV
4. Проверяет целостность данных (нет пустых полей)
5. API endpoint для интеграции

Пример входных данных (прикрепить):
[вставить пример ответа AI]

Ожидаемый результат (прикрепить):
[вставить желаемый Excel/CSV]

Срок: [дата]"

✅ Чеклист принятия результата

  • Формат соответствует запросу? (таблица / JSON / чеклист)
  • Все поля заполнены? (нет пропусков)
  • Данные корректны? (проверьте 2-3 случайные ячейки)
  • Можно скопировать в Excel / 1С / CRM?
  • Структура понятна без объяснений?
  • Единицы измерения указаны? (рубли, дни, проценты)

🎯 Главное из урока

Указывайте формат в промпте. Таблица для сравнения, JSON для автоматизации, чеклист для проверки. Это экономит 5-10 минут на форматирование каждый раз. За месяц — экономия 5-10 часов.
