8.3 · Планирование работ и ресурсов: графики, люди, техника

🎯 Без кода · ✅ Интерактивный квиз

🎯 Зачем это бизнесу

Организатор работ (прораб, начальник участка) ежедневно решает: кто работает, на чём, в какой последовательности, что задерживается. AI помогает строить графики, оптимизировать ресурсы, прогнозировать задержки.

Кейс: AI проанализировал 3-месячный график и предупредил: бетонная работа на 5-м этаже начнётся раньше, чем кран освободится с 4-го. Задержка: 2 дня. Организатор перенёс начало на 2 дня — срыва графика не было.

📚 Что изучаем

Что AI делает для организатора работ

  • 📅 Графики: Построение / корректировка сетевого графика, критический путь
  • 👷 Ресурсы: Распределение бригад, техники, материалов по объектам
  • ⚠️ Риски: Предупреждение о задержках, перегрузках, конфликтах ресурсов
  • 📊 Отчётность: Журнал работ, акты выполненных работ, сравнение план/факт
  • 🔮 Прогноз: Когда закончится объект при текущем темпе?
  • 📋 Наряд-задания: Автоматическая генерация на основе графика

Входные данные для AI

ДанныеФорматЧто AI анализирует
Сетевой график (MS Project, Excel).mpp, ExcelКритический путь, ресурсы, зависимости
Журнал работ (ежедневные отчёты)PDF, Excel, текстПлан/факт, % выполнения, задержки
Сметы / локальные сметыPDF, ExcelОбъёмы, трудоёмкость, длительность
Наличие ресурсов (бригады, техника)Excel, 1С, SAPЗагрузка, конфликты, доступность

Пример запроса к AI

Промпт для графика
"Проанализируй график строительства (файл приложен).

1. Найди критический путь (работы, которые нельзя задержать)
2. Проверь загрузку ресурсов: есть ли перегрузки (2 бригады на 1 объект)?
3. Проверь зависимости: начинается ли работа раньше, чем закончится предыдущая?
4. Сравни с погодным прогнозом: какие наружные работы попадают на дожди?
5. Прогноз: дата окончания при текущем темпе?

Выведи:
- Таблица рисков: Работа | Риск | Уровень | Рекомендация
- Обновлённый график с корректировками
- Список наряд-заданий на следующую неделю"

🛠️ Как применить

Сценарий: еженедельный обзор графика

Процесс
1. Загружаем в AI: график (MS Project/Excel), журнал работ за неделю, ресурсы
2. Запрос: "Проверь график. Критический путь? Риски? Загрузка ресурсов?"
3. AI анализирует и выдаёт:
   - Критический путь (выделен красным)
   - Риски: перегрузка крана 15.03, бетон придёт позже
   - Рекомендации: перенести работы, добавить бригаду
4. Прораб корректирует график
5. AI генерирует наряд-задания на следующую неделю
6. Раздаёт бригадирам

Сравнение план/факт

AI ежедневно сравнивает:

  • Запланировано объёмов vs выполнено
  • Даты начала/окончания vs факт
  • Расход материалов vs нормы
  • Трудозатраты vs план

При отклонении > 10% — предупреждение с рекомендацией.

👨‍💻 Как делегировать разработчику

Шаблон задачи

"Нужен AI-инструмент для управления строительными графиками.

Входные данные:
- График: MS Project / Excel (сетевой график)
- Журнал работ: ежедневные отчёты (PDF, Excel, текст)
- Ресурсы: бригады, техника, материалы (Excel, 1С, SAP)
- Сметы: локальные сметы (PDF, Excel)
- Погода: API (или Excel с прогнозом)

Что должен делать:
1. Анализировать критический путь
2. Проверять загрузку ресурсов (конфликты, перегрузки)
3. Сравнивать план/факт
4. Прогнозировать дату окончания
5. Генерировать наряд-задания
6. Предупреждать о рисках (задержки, погода, ресурсы)

Выход:
- PDF-отчёт (еженедельный)
- Excel-таблица (обновлённый график)
- Email/SMS-уведомления (критичные риски)

Интеграция: [MS Project / 1С / SAP / Excel]
Срок: [дата]"

✅ Чеклист принятия результата

  • Критический путь корректен? (сверить с ручным расчётом)
  • Риски реальны? (проверить 2-3 на графике)
  • Загрузка ресурсов адекватна? (нет перегрузок > 100%)
  • Прогноз даты окончания реалистичен?
  • Наряд-задания соответствуют графику?
  • Формат удобен для ежедневного использования?

🎯 Главное из урока

AI — прогнозный аналитик для стройки. Он видит риски, которые человек пропускает: перегрузку ресурсов, конфликты в графике, погодные задержки. Не заменяет прораба, но даёт ему инструмент для принятия решений на основе данных, а не на ощущениях.
