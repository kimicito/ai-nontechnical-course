1.5 · Практика: автоматизируй свою первую задачу

🎯 Практика · ✅ Без кода

🎯 Зачем это бизнесу

Теория без практики бесполезна. Выберите одну реальную задачу из вашей работы и автоматизируйте её за 1 час.

Цель: После этого урока у вас будет работающий промпт для реальной задачи, который экономит 2-3 часа в неделю.

📚 Шаг 1: Выберите задачу

Идеальная задача для автоматизации

  • ✅ Повторяется > 3 раз в неделю
  • ✅ Занимает > 20 минут каждый раз
  • ✅ Есть чёткий алгоритм (можно описать шаги)
  • ✅ Не требует критичных решений (безопасно ошибиться)

Примеры задач по ролям

🛒 Закупщик

"Сравни 3 коммерческих предложения и сделай рекомендацию"

"Проверь договор на риски перед подписанием"

"Составь отчёт о закупках за неделю из CSV"

📊 Аналитик

"Проанализируй продажи за месяц и найди аномалии"

"Составь прогноз на следующий квартал на основе трендов"

⚖️ Юрист

"Проверь 5 договоров на типичные риски"

"Сравни новый договор с корпоративным шаблоном"

📦 Логист

"Оптимизируй маршрут доставки по 10 точкам"

"Рассчитай стоимость доставки разными способами"

🛠️ Шаг 2: Алгоритм автоматизации

Шаг 1: Опишите задачу

Что делаете, зачем, как часто, сколько времени занимает.

Пример
Задача: Сравнение КП поставщиков
Частота: 5 раз в неделю
Время: 1.5 часа каждый раз
Проблема: Долго, легко пропустить важную деталь

Шаг 2: Напишите промпт (RCOF)

Роль, Контекст, Цель, Формат.

Пример
Ты — опытный закупщик FMCG.
Выбираем поставщика картонной упаковки для печенья.
Получили 3 предложения.

Задача: сравни предложения по критериям и рекомендуй лучшее.

Критерии:
1. Цена за единицу (включая доставку)
2. Минимальный заказ
3. Срок изготовления
4. Условия оплаты

Формат: таблица + рекомендация с обоснованием.
Ограничение: срок > 14 дней = риск.

Шаг 3: Проверьте на 3 примерах

Возьмите 3 реальных примера и проверьте, работает ли промпт.

Что проверять
  • Результат соответствует запросу?
  • Формат удобен?
  • Нет ошибок/галлюцинаций?
  • Время сократилось?

Шаг 4: Запишите процесс

Создайте шаблон в Notion / Word / Excel.

Структура шаблона
Название задачи: [что делаете]
Частота: [сколько раз]
Время до: [X минут]
Время после: [Y минут]

ПРОМПТ:
[вставить промпт]

ПРИМЕР ВХОДНЫХ ДАННЫХ:
[вставить пример]

ПРИМЕР РЕЗУЛЬТАТА:
[вставить результат]

ПРОВЕРЕНО: [дата]
РАБОТАЕТ: Да / Нет (что исправить)

Шаг 5: Поделитесь с коллегами

Покажите, как экономите время. Это мотивирует других.

Как поделиться
  • Отправьте шаблон промпта в корпоративный чат
  • Покажите на совещании: "Я сократил время на X часов"
  • Создайте базу знаний с шаблонами для команды

📝 Проверка понимания

Вопрос 1: Какая задача подходит для автоматизации AI?

A) Разовая задача, которую делаем раз в год
B) Повторяется >3 раз в неделю, занимает >20 минут, есть чёткий алгоритм
C) Творческая задача без чётких критериев

Вопрос 2: Что такое RCOF в промпте?

A) Название модели
B) Роль, Контекст, Цель, Формат — структура промпта
C) Размер файла в байтах

Вопрос 3: Сколько примеров нужно для проверки промпта?

A) Минимум 3 реальных примера
B) Достаточно 1 примера
C) Не нужно проверять, AI всегда прав

📝 Практическое задание

Задание: Автоматизируйте одну задачу за 60 минут

  1. Минуты 0-10: Выберите задачу из списка выше (или свою)
  2. Минуты 10-25: Напишите промпт по формуле RCOF
  3. Минуты 25-40: Проверьте на 3 реальных примерах
  4. Минуты 40-50: Доработайте промпт (что не работает?)
  5. Минуты 50-60: Запишите шаблон и сохраните

Критерии приёмки

  • Промпт содержит RCOF
  • Проверен на 3 примерах
  • Результат стабильный (не меняется от запроса к запросу)
  • Экономия времени > 50%
  • Шаблон записан и доступен для повторного использования

🎯 Главное из урока

Лучший способ научиться — сделать. Выберите одну задачу, напишите промпт, проверьте на 3 примерах, запишите процесс. Экономия 2-3 часа в неделю — реальность, а не теория. Поделитесь результатом с коллегами — это ускорит внедрение AI в компании.
