16.2 Внешние задачи: внешний ИИ

⏱ 20 мин · для комплаенс-офицеров
Открытые данные (внешний ИИ)
Все задачи этого урока — с публичными санкционными списками, нормативными актами, открытыми фреймворками, типовыми политиками. Можно использовать Claude, ChatGPT, Gemini.

Что можно делать во внешнем ИИ

Внешний ИИ безопасен там, где работа идёт с нормативным полем как таковым — какие списки существуют, что требует регулятор, как устроена методология — до того, как это применяется к реальному имени или сделке.

Главное правило: если в запросе нет конкретных имён, названий компаний, сумм, номеров счетов, дат сделок — можно использовать внешний ИИ.

1. Мониторинг санкционных режимов

Суммаризация изменений в санкционных списках OFAC, EU, UK, ООН и национальных регуляторов.

Пример промпта (внешний ИИ + веб-поиск)
Проанализируй изменения санкционных режимов за последний месяц:

Юрисдикции: OFAC (США), EU, UK, ООН

Сделай:
1. Новые лица/организации, добавленные в списки
2. Изменения в секторальных санкциях
3. Новые ограничения по товарам/технологиям
4. Разъяснения регуляторов (guidance updates)
5. Тренды (на что обращать внимание)
6. Рекомендации для компаний в целом

Формат: аналитическая записка (1-2 страницы)

Результат: обзор изменений, экономит 3–4 часа.

2. Заключения по экспортному контролю (общие)

Анализ применимости норм экспортного контроля в общем виде, без привязки к конкретной сделке.

Пример промпта
Объясни применимость экспортного контроля:

Тема: [УКАЖИТЕ: dual-use товары/ограничения на технологии/военное назначение]
Юрисдикции: [УКАЖИТЕ: США/ЕС/РФ/КНР]

Сделай:
1. Применимые режимы экспортного контроля
2. Категории товаров/технологий
3. Лицензионные требования
4. Исключения и освобождения
5. Процедура получения лицензии
6. Практические рекомендации (общие)
7. Ссылки на нормативные акты

Важно: без привязки к конкретной сделке компании.

Результат: справка по экспортному контролю, экономит 2–3 часа.

3. Разработка комплаенс-политик по типовым фреймворкам

Создание политик на основе международных стандартов (ISO 37001, UK Bribery Act, FCPA).

Пример промпта
Разработай комплаенс-политику на основе международных стандартов:

Тип политики: [УКАЖИТЕ: антикоррупционная/AML/санкционная/конфликт интересов]
Базовый фреймворк: [УКАЖИТЕ: ISO 37001/UK Bribery Act/FCPA guidance]

Сделай:
1. Структура политики (разделы)
2. Основные положения (принципы)
3. Обязанности сотрудников (по уровням)
4. Процедуры документирования
5. Порядок расследования нарушений
6. Санкции за нарушения
7. Периодичность пересмотра
8. Приложения (формы, чек-листы)

Формат: готовый документ для адаптации под компанию.

Результат: черновик политики, экономит 4–5 часов.

4. Кодекс деловой этики

Разработка общей части кодекса на основе лучших практик.

Пример промпта
Разработай кодекс деловой этики:

Компания: крупная производственная компания (B2B)
География: международные операции

Сделай:
1. Ценности компании (5-7 пунктов)
2. Поведение в отношениях с контрагентами
3. Правила приёма подарков и гостеприимства
4. Конфликт интересов (определение, примеры)
5. Защита активов и информации
6. Трудовая этика (дискриминация, домогательства)
7. Обязанность сообщать о нарушениях
8. Порядок расследования
9. Ответственность

Тон: понятный, конкретный примеры, без абстракций.

Результат: готовый кодекс, экономит 3–4 часа.

5. Обучающие программы по комплаенсу

Подготовка обучающих материалов общего содержания.

Пример промпта
Разработай обучающий курс по комплаенсу:

Тема: [УКАЖИТЕ: AML/антикоррупция/санкции/конфликт интересов]
Аудитория: [УКАЖИТЕ: все сотрудники/менеджеры/финансовый отдел]
Формат: 10 слайдов + тест (5 вопросов)

Составь:
1. Цели обучения
2. Основные понятия (простым языком)
3. Обязанности сотрудника
4. Красные флаги (примеры)
5. Что делать при подозрении
6. Тест (5 вопросов с вариантами)
7. Чек-лист для самопроверки

Тон: доступный, с примерами из жизни.

Результат: готовый материал для тренинга, экономит 3–4 часа.

6. Бенчмаркинг комплаенс-практик отрасли

Сравнение комплаенс-практик с международными холдингами по публичным данным.

Пример промпта (веб-поиск)
Проведи бенчмаркинг комплаенс-практик:

Отрасль: [УКАЖИТЕ]
Параметры:
- Структура комплаенс-функции
- Использование технологий (автоматизация скрининга)
- Программа обучения (часы на сотрудника)
- Whistleblower-каналы
- Внешние аудиты (частота)
- Публичная отчётность (ESG/anti-bribery reports)

Сделай:
1. Обзор практик 3-5 международных компаний
2. Сравнительная таблица
3. Лучшие практики (best practices)
4. Рекомендации по улучшению
5. Тренды в комплаенс-функции

Источники: публичные отчёты, ESG-раскрытия, кодексы этики.

Результат: обзор отраслевых практик, экономит 4–5 часов.

Проверка знаний

Вопрос 1: Можно ли во внешнем ИИ проверить конкретного контрагента по санкционным спискам?

Да, списки публичны
Нет, сам факт "кого проверяли" — чувствительная информация
Только если обезличить имя

Вопрос 2: Разработка комплаенс-политики по ISO 37001 — какой контур?

Внешний ИИ — фреймворк публичен
On-prem — политика внутренняя
Гибрид — структура в облаке, адаптация внутри

Вопрос 3: Бенчмаркинг комплаенс-практик — какой контур?

Внешний ИИ — используются публичные отчёты
On-prem — сравнение с конкурентами
Гибрид — публичные данные + внутренняя аналитика
