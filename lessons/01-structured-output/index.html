1.2 · Как получить ответ от AI в нужном формате

🎯 Без кода · ✅ Интерактивный квиз

🎯 Зачем это бизнесу

AI по умолчанию отвечает сплошным текстом. Вы получаете абзац, который приходится перебивать руками в Excel или Word.

Проблема: Менеджер получает анализ поставщиков текстом, тратит 15 минут на перенос в таблицу.

Решение: Сразу попросить AI выдать результат таблицей. Экономия: 15 минут × 20 задач = 5 часов в неделю.

📚 Что изучаем

4 формата ответа для менеджера

📊 Таблица

Для сравнения, анализа, отчётов. Копируется в Excel одним кликом.

☑️ Список с галочками

Для проверок, аудита, планов действий. Понятно и наглядно.

📧 Готовое письмо

AI сразу пишет письмо поставщику, клиенту или руководству.

📋 По шагам (план)

Чёткий порядок действий с датами и ответственными.

Как попросить таблицу

💡 Пример промпта
Ты — финансовый аналитик.
Проанализируй 3 инвестиционных проекта (данные ниже).

Выведи результат в виде таблицы:
| Проект | Окупаемость | Риск | Рекомендация |

После таблицы — краткое обоснование.

Как попросить список с галочками

💡 Пример промпта
Ты — аудитор.
Проверь договор поставки на риски.

Выведи результат списком:
☐ Риск: [описание] — Уровень: [Высокий/Средний/Низкий] — Что делать: [рекомендация]

В конце: общий вывод — подписывать договор или нет.

🛠️ Как применить

Сценарий 1: Сравнение поставщиков

Задача: Выбрать поставщика упаковки из 3 вариантов.

Промпт
Ты — закупщик FMCG.
Сравни 3 предложения по критериям и выведи таблицей:
| Критерий | Поставщик А | Поставщик Б | Поставщик В | Лидер |
| Цена за ед. | | | | |
| Мин. заказ | | | | |
| Срок доставки | | | | |
| Условия оплаты | | | | |

Дай рекомендацию: кого выбрать и почему.

Результат: Готовая таблица для презентации руководству. Копируется в Excel одним кликом.

Сценарий 2: Аудит договора

Задача: Проверить договор на риски перед подписанием.

Промпт
Ты — корпоративный юрист.
Проанализируй договор поставки на риски.

Выведи результат списком:
☐ Риск: [описание] — Уровень: [Высокий/Средний/Низкий] — Рекомендация: [что сделать]

В конце: общий вывод — приемлем ли договор.

Сценарий 3: Подготовка письма

Задача: Написать письмо поставщику с претензией по качеству.

Промпт
Ты — менеджер по закупкам.
Напиши деловое письмо поставщику.

Контекст: в последней партии брак 15%, что превышает допустимые 3%.

Формат: готовое письмо в деловом стиле.
Структура:
1. Ссылка на договор и допустимый процент брака
2. Факты (партия, дата, количество, фото во вложении)
3. Требование: замена или возврат средств
4. Срок ответа: 5 рабочих дней

Тон: твёрдый, но корректный. Без угроз.

Результат: Готовое письмо, которое можно сразу отправить. Осталось только вставить реквизиты.

Сценарий 4: План действий

Задача: Составить план внедрения нового процесса.

Промпт
Ты — руководитель проекта.
Составь план внедрения CRM-системы.

Формат: пошаговый план с датами:
1. Шаг — Срок — Ответственный — Результат
2. Шаг — Срок — Ответственный — Результат
...

Всего должно быть 8-10 шагов.
Срок реализации: 3 месяца.
Бюджет: до 500 000 руб.

Результат: Готовый план для утверждения руководством. Копируется в Word или Excel.

👨‍💻 Как делегировать разработчику

Шаблон задачи

"Нужен инструмент, который автоматически превращает ответы AI в Excel-таблицы.

Требования:
1. Вставляешь текст от AI — получаешь готовую таблицу .xlsx
2. Работает с любыми таблицами (сравнение поставщиков, отчёты, планы)
3. Сохраняет форматирование (жирный шрифт для заголовков, цвета)
4. Простой интерфейс: копировать-вставить — готово
5. Не требует программирования от пользователя

Пример входных данных (прикрепить):
[вставить пример ответа AI]

Ожидаемый результат (прикрепить):
[вставить желаемый Excel]

Срок: [дата]"

✅ Чеклист принятия результата

  • Формат соответствует задаче? (таблица / список / письмо / план)
  • Все данные на месте? (нет пропусков)
  • Можно скопировать в Excel / Word / Outlook?
  • Структура понятна без объяснений?
  • Единицы измерения указаны? (рубли, дни, проценты)
  • Проверили 2-3 случайных пункта на корректность?

🎯 Главное из урока

Всегда указывайте формат ответа в промпте. Таблица для сравнения, список для проверок, письмо для коммуникации, план для действий. Это экономит 10-15 минут на каждой задаче. За месяц — экономия 5-10 часов.
