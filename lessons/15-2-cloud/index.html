15.2 Внешние задачи: внешний ИИ

⏱ 20 мин · для юристов, compliance-специалистов
Открытые данные (внешний ИИ)
Все задачи этого урока — с типовыми шаблонами, публичным законодательством, открытыми судебными решениями, обучающими материалами. Можно использовать Claude, ChatGPT, Gemini.

Что можно делать во внешнем ИИ

В отличие от задач с конкретными делами и контрагентами, работа с открытым правовым полем безопасна и эффективна:

  • Составление типовых договоров по шаблонам (без реальных условий)
  • Мониторинг изменений законодательства (публичные источники)
  • Анализ судебной практики (открытые решения)
  • Патентный поиск (открытые базы)
  • Обучающие материалы (общего характера)
  • Бенчмаркинг юридических практик (публичные данные)
Главное правило: если в запросе нет названий компаний, сумм, конкретных дат, персональных данных, стратегических планов — можно использовать внешний ИИ.

1. Типовые шаблоны договоров

Составление структуры типового договора без реальных условий — публичная методология.

Пример промпта (внешний ИИ)
Подготовь типовой договор поставки:

Параметры (без конкретных условий):
- Тип: договор поставки товара
- Правовая система: [УКАЖИТЕ СТРАНУ]
- Сторона покупатель: крупная компания (B2B)
- Сторона поставщик: производитель
- Объект: [УКАЖИТЕ ТИП ТОВАРА без количества/цен]

Составь:
1. Преамбула (типовая)
2. Предмет договора (структура описания)
3. Цена и порядок расчётов (типовые формулировки)
4. Поставка и передача рисков (Incoterms — типовые)
5. Качество и приёмка (типовые условия)
6. Ответственность сторон (без конкретных штрафов)
7. Форс-мажор (типовая статья)
8. Порядок разрешения споров (типовые варианты)
9. Заключительные положения

Формат: для каждой статьи — базовый текст + альтернативы

Результат: типовой шаблон для юротдела, экономит 3–4 часа.

2. Мониторинг изменений законодательства

Суммаризация изменений в законодательстве по конкретным темам.

Пример промпта (веб-поиск)
Проанализируй изменения законодательства за последние 6 месяцев:

Темы:
1. [УКАЖИТЕ ОТРАСЛЬ: корпоративное право/налоги/трудовое/антимонопольное]
2. [УКАЖИТЕ ВТОРУЮ ТЕМУ]
3. [УКАЖИТЕ ТРЕТЬЮ ТЕМУ]

Юрисдикция: [УКАЖИТЕ СТРАНУ/РЕГИОН]

Сделай:
1. Список ключевых изменений (дата, номер документа, суть)
2. Анализ влияния на бизнес (общий, без привязки к конкретной компании)
3. Сроки вступления в силу
3. Необходимые действия компаний в целом
4. Рекомендации по подготовке

Формат: аналитическая записка (1-2 страницы)

Результат: обзор изменений, экономит 4–5 часов.

3. Анализ судебной практики

Поиск и анализ открытых судебных решений по аналогичным вопросам.

Пример промпта (веб-поиск)
Проанализируй судебную практику по вопросу:

Вопрос: [УКАЖИТЕ ПРАВОВОЙ ВОПРОС без привязки к конкретному делу]
Юрисдикция: [УКАЖИТЕ]
Период: последние 2 года

Сделай:
1. Подборка релевантных дел (5-10 решений)
2. Краткое содержание каждого дела
3. Позиция суда (ключевые выводы)
4. Тренды в практике (как меняется подход судов)
5. Риски для компаний в целом
6. Рекомендации по минимизации рисков

Важно: только публичные решения, без упоминания конкретных компаний если не является публичным делом.

Результат: обзор практики, экономит 5–6 часов.

4. Патентный поиск и анализ патентной чистоты

Поиск по открытым патентным базам, анализ свободы действий.

Пример промпта (веб-поиск)
Проведи патентный поиск по технологии:

Технология: [УКАЖИТЕ ОБЛАСТЬ без конкретных деталей продукта]
Ключевые слова: [УКАЖИТЕ 3-5 ТЕРМИНОВ]
Классы МПК: [УКАЖИТЕ если известны]
Юрисдикции: [УКАЖИТЕ: РФ, США, ЕПВ, КНР и т.д.]

Сделай:
1. Поиск релевантных патентов (5-10)
2. Анализ каждого патента (что охраняется, срок, статус)
3. Анализ патентной чистоты (есть ли препятствия)
4. Рекомендации по обходным решениям (если есть)
5. Рекомендация по дальнейшим действиям

Важно: это предварительный анализ, не заменяет патентного поверенного.

Результат: предварительный анализ, экономит 3–4 часа (но не заменяет патентного поверенного).

5. Обучающие материалы для сотрудников

Подготовка обучающих материалов по правовым темам общего характера.

Пример промпта
Подготовь обучающий материал для сотрудников:

Тема: [УКАЖИТЕ: коммерческая тайна/антикоррупция/конкуренция/персональные данные]
Аудитория: [УКАЖИТЕ: все сотрудники/менеджеры/продавцы]
Формат: презентация (10 слайдов) + тест (5 вопросов)

Составь:
1. Цели обучения (что должен знать сотрудник)
2. Основные положения (простым языком)
3. Примеры ситуаций (гипотетические, не реальные)
4. Правила поведения (do's and don'ts)
5. Ответственность за нарушение
6. Тест (5 вопросов с вариантами ответов)
7. Чек-лист для самопроверки

Тон: доступный, без юридического жаргона.

Результат: готовый материал для тренинга, экономит 3–4 часа.

6. Бенчмаркинг юридических практик отрасли

Сравнение юридических процессов с аналогичными компаниями по публичным данным.

Пример промпта (веб-поиск)
Проведи бенчмаркинг юридических практик в отрасли:

Отрасль: [УКАЖИТЕ]
Параметры для сравнения:
- Структура юридического департамента
- Использование legal tech
- Внешние юристы vs in-house
- Судебная нагрузка (типичные споры)
- Стоимость юридических услуг (публичные данные)
- Compliance-практики

Источники: отраслевые отчёты, публикации, конференции

Сделай:
1. Обзор практик 3-5 компаний отрасли (публичные данные)
2. Сравнительная таблица
3. Лучшие практики (best practices)
4. Рекомендации по оптимизации юридической функции

Важно: только публичные данные, без конкретных сумм контрактов.

Результат: обзор отраслевых практик, экономит 4–5 часов.

Проверка знаний

Вопрос 1: Можно ли составить договор поставки с конкретной ценой во внешнем ИИ?

Да, если это типовой договор
Нет, конкретная цена — коммерческая тайна и идёт в on-prem
Да, но только для компаний небольшого размера

Вопрос 2: Анализ судебной практики — какой контур?

Внешний ИИ — судебные решения публичны
On-prem — судебная практика конфиденциальна
Гибрид — обезличить названия компаний

Вопрос 3: Патентный поиск — какой контур?

Внешний ИИ — патентные базы открыты
On-prem — это интеллектуальная собственность компании
Гибрид — базы снаружи, анализ внутри
