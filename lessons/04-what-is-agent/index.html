4.1 · Что такое AI-агент: от чата к автономности

⏱️ 40 минут · 🎯 Без кода · ✅ Интерактивный квиз

🎯 Зачем это бизнесу

Обычный AI ждёт ваших команд. Агент действует сам: ставит цели, планирует шаги, выполняет, проверяет результат.

Пример: Вы говорите агенту "Подготовь отчёт о рынке упаковки". Агент сам:

  1. Ищет информацию в интернете
  2. Анализирует найденное
  3. Составляет таблицу сравнения
  4. Пишет выводы и рекомендации
  5. Отправляет вам готовый документ

Вы потратили: 5 минут на постановку задачи. Агент потратил: 20 минут на выполнение.

📚 Что изучаем

Чат-бот vs Агент

💬 Чат-бот

  • Ждёт вопроса
  • Отвечает один раз
  • Не запоминает контекст
  • Не выполняет действия
  • Требует постоянного внимания

🤖 Агент

  • Ставит цели сам
  • Выполняет цепочку действий
  • Запоминает состояние
  • Использует инструменты
  • Работает автономно

Компоненты агента

КомпонентЧто делаетАналогия
Мозг (LLM)Думает, планирует, принимает решенияГолова сотрудника
ПамятьЗапоминает контекст, историю, фактыБлокнот сотрудника
ИнструментыВыполняет действия: поиск, email, ExcelРуки сотрудника
ПланировщикРешает, что делать следующим шагомМенеджер проекта

🛠️ Как применить

Примеры бизнес-агентов

🕵️ Агент-аналитик рынка

Цель: Еженедельный отчёт о конкурентах

Действия: Ищет цены конкурентов → Анализирует → Составляет таблицу → Отправляет отчёт

Экономия: 4 часа в неделю

📋 Агент-помощник закупщика

Цель: Обработка входящих КП

Действия: Получает письмо → Извлекает данные → Сравнивает с базой → Формирует рекомендацию → Отправляет закупщику

Экономия: 2 часа в день

🔍 Агент-аудитор

Цель: Проверка договоров на соответствие стандартам

Действия: Получает договор → Сравнивает с шаблоном → Находит отклонения → Формирует отчёт → Отправляет юристу

Экономия: 3 часа на документ

👨‍💻 Как делегировать разработчику

Шаблон задачи

"Нужен AI-агент для [задача].

Цель агента: [что должен достичь]

Входные данные: [откуда получает информацию]
Выходные данные: [что должен produce]

Инструменты, которые нужны:
- [поиск в интернете / email / Excel / API / etc.]

Ограничения:
- [бюджет на API-запросы]
- [время выполнения]
- [что требует подтверждения человека]

Пример задачи и ожидаемого результата: [привести]

Срок: [дата]"

✅ Чеклист принятия результата

  • Агент достигает цели полностью? (от начала до конца)
  • Не зацикливается на одном шаге?
  • Использует инструменты корректно?
  • Запоминает контекст между шагами?
  • Укладывается в бюджет и время?
  • Есть "человек в цикле" для критичных решений?

📝 Проверка понимания

Вопрос 1: Чем агент отличается от чат-бота?

A) Агент сам ставит цели и выполняет действия B) Агент дороже C) Агент быстрее отвечает

Вопрос 2: Что такое "память" агента?

A) Способность запоминать контекст и историю B) Объём жёсткого диска C) Скорость работы

Вопрос 3: Зачем агенту "инструменты"?

A) Чтобы выполнять действия: искать, отправлять, записывать B) Чтобы выглядеть сложнее C) Это необязательно

🎯 Главное из урока

Агент = автономный сотрудник. У него есть мозг (LLM), память, инструменты и планировщик. Он не просто отвечает на вопросы — он решает задачи от начала до конца. Начните с простых агентов, добавляйте сложность постепенно. Всегда оставляйте "человека в цикле".
