7.1 · Поиск и анализ поставщиков: от поиска до короткого списка

🎯 Без кода · ✅ Интерактивный квиз

🎯 Зачем это бизнесу

Поиск поставщиков — рутинная, но критичная задача. Закупщик тратит часы на ручной поиск по сайтам, ЭТП, каталогам. AI-агент может делать это автоматически: ходить по сайтам, извлекать контакты, цены, условия — и формировать короткий список кандидатов.

Экономия: 8 часов ручного поиска → 30 минут на проверку результатов AI. Освобождается время для переговоров и анализа.

📚 Что изучаем

Как AI ищет поставщиков

AI-агент может:

  • 🔍 Искать по сайтам (парсинг каталогов, поисковая выдача)
  • 📋 Извлекать: название, контакты, ассортимент, цены, условия доставки
  • 🌍 Работать с международными площадками (Alibaba, ThomasNet, Europages)
  • ⚡ Фильтровать по критериям (РФ/импорт, MOQ, сертификаты)
  • 📊 Формировать сравнительную таблицу

Фильтры для AI-агента

Промпт для агента-поисковика
"Найди поставщиков [наименование товара] со следующими критериями:

Обязательно:
- Работают в РФ (или импорт из [страна])
- MOQ ≤ [количество]
- Есть сертификат ISO / ГОСТ Р
- Опыт работы > 2 лет

Желательно:
- Склад в [городе]
- Отсрочка платежа ≥ 30 дней
- Положительные отзывы

Выведи таблицу: Компания | Контакты | Цена | MOQ | Условия | Примечания"

🛠️ Как применить

Сценарий: поиск поставщика станков

Шаг за шагом
1. Отправляем AI-агенту ТЗ на станок (мощность, размеры, точность)
2. Агент ищет 3 источника: ЭТП, B2B-каталоги, сайты производителей
3. Агент извлекает данные и формирует таблицу
4. Закупщик проверяет ТОП-5 кандидатов
5. Отправляет запросы КП вручную (или через AI-рассылку)

Результат: 3 часа поиска → 15 минут проверки

Что проверять вручную

  • ⚠️ Актуальность контактов (AI может найти устаревшие)
  • ⚠️ Статус компании (не в процессе ликвидации)
  • ⚠️ Реальность цен (иногда на сайтах — "от...")
  • ⚠️ Отзывы и репутация

👨‍💻 Как делегировать разработчику

Шаблон задачи

"Нужен AI-агент для поиска поставщиков.

Входные данные:
- Наименование товара / ТЗ
- Критерии отбора (MOQ, география, сертификаты)
- Источники для поиска: [список]

Что должен делать агент:
1. Искать по заданным источникам
2. Извлекать: название, контакты, цены, условия
3. Фильтровать по критериям
4. Выводить таблицу сравнения
5. Флагом отмечать "требует проверки"

Интеграции:
- Выгрузка в Excel / 1С / SAP
- Сохранение в базу поставщиков

Срок: [дата]"

✅ Чеклист принятия результата

  • Найдено ≥ 5 релевантных поставщиков?
  • Контакты актуальны? (проверить 2-3)
  • Цены соответствуют рынку? (сверить с аналогами)
  • Фильтры работают корректно?
  • Результат в удобном формате (Excel/таблица)?
  • Есть пометки "требует проверки"?

🎯 Главное из урока

AI-агент — ваш разведчик. Он не заменяет закупщика, но освобождает от рутины поиска. Задаёте критерии — получаете короткий список. Проверяете контакты и репутацию — принимаете решение. Экономия: 80% времени на поиске.
