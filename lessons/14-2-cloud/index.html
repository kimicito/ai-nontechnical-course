Уровень 14 — Продажи и маркетинг

14.2 Внешние задачи: внешний ИИ

⏱ 20 мин · для маркетологов, SMM-менеджеров, контент-менеджеров
Открытые данные (внешний ИИ)
Все задачи этого урока — с публичным контентом, открытыми источниками, общими методологиями. Можно использовать Claude, ChatGPT, Gemini.

Почему маркетинг — одна из самых безопасных зон для внешнего ИИ

Маркетинговый контент по определению публичен. Конечный продукт (пост, брошюра, рекламный текст) предназначен для широкой аудитории. Риск утечки коммерческой тайны минимален — если не передавать внутренние бюджеты, цены и стратегические планы.

Важно: перед отправкой в облако — проверь, что в запросе нет реальных цен, бюджетов, клиентских данных, стратегических планов. Если есть — обезличь или отправь в on-prem.

1. Разработка контент-плана

Генерация идей, рубрик, календаря публикаций — родная территория genAI.

Пример промпта (Claude с веб-поиском)
Подготовь контент-план на месяц для B2B-компании в сфере [УКАЖИТЕ ОТРАСЛЬ]:

Параметры:
- Каналы: LinkedIn, корпоративный блог, email-рассылка
- Целевая аудитория: [УКАЖИТЕ: закупщики, инженеры, директора]
- Тематические блоки: экспертиза, кейсы, тренды, продукт
- Частота: 3 поста в неделю LinkedIn, 2 статьи в месяц, 4 письма

Сделай:
1. Темы на каждый пост (заголовок + формат: карусель, текст, видео)
2. Календарь публикаций
3. Хуки для каждой темы (что привлечёт внимание)
4. Призывы к действию (CTA)
5. Перекрёстные ссылки между каналами

Результат: готовый контент-план, экономит 4–5 часов.

2. Копирайтинг рекламных кампаний

Генерация вариантов заголовков, текстов объявлений, описаний — без реальных бюджетов.

Пример промпта
Напиши 5 вариантов рекламного объявления для B2B-аудитории:

Продукт: [УКАЖИТЕ ТИП ПРОДУКТА] для [УКАЖИТЕ ОТРАСЛЬ]
УТП: [УКАЖИТЕ КЛЮЧЕВОЕ ПРЕИМУЩЕСТВО]
ЦА: [УКАЖИТЕ ДОЛЖНОСТЬ], компании [УКАЖИТЕ РАЗМЕР]
Формат: Google Ads (3 заголовка по 30 символов, 2 описания по 90)

Дополнительно:
- 3 варианта для LinkedIn (более экспертный тон)
- 3 варианта для email-рассылки (персонализированные)
- A/B-тест гипотезы: что тестировать (заголовок, CTA, оффер)

Результат: набор объявлений для тестов, экономит 2–3 часа.

3. SEO-оптимизация контента

Генерация метаописаний, заголовков H1-H3, alt-текстов — публичный контент.

Пример промпта
Оптимизируй текст для SEO:

Исходный текст (статья в блог):
[ВСТАВЬТЕ ТЕКСТ СТАТЬИ]

Ключевые слова (приоритетные):
1. [УКАЖИТЕ КЛЮЧЕВОЕ СЛОВО 1]
2. [УКАЖИТЕ КЛЮЧЕВОЕ СЛОВО 2]
3. [УКАЖИТЕ КЛЮЧЕВОЕ СЛОВО 3]

Сделай:
1. Заголовок H1 (60 символов, с ключом)
2. Мета-описание (150 символов, с CTA)
3. Структуру H2-H3 (с вхождением ключей)
4. Alt-тексты для 3 изображений
5. Чек-лист внутренней оптимизации (перелинковка)

Результат: оптимизированный текст, экономит 1.5 часа.

4. Анализ конкурентов по открытым данным

Продуктовая линейка, позиционирование, публичные цены — всё это в открытом доступе.

Пример промпта (веб-поиск)
Проведи анализ 3 конкурентов в сегменте [УКАЖИТЕ РЫНОК]:

Конкуренты:
1. [УКАЖИТЕ КОМПАНИЮ 1]
2. [УКАЖИТЕ КОМПАНИЮ 2]
3. [УКАЖИТЕ КОМПАНИЮ 3]

Данные для сбора (публичные источники):
- Продуктовая линейка (категории, SKU)
- Позиционирование (как себя позиционируют)
- Публичные цены (если есть на сайте)
- Каналы продвижения (где рекламируются)
- Сильные стороны (по отзывам, кейсам)
- Слабые стороны (жалобы, пробелы)

Сделай:
1. Сравнительную таблицу
2. Карту позиционирования (как они себя видят vs как видят клиенты)
3. Белые пятна (что не покрывают конкуренты — наша возможность)
4. Рекомендации по дифференциации

Результат: обзор конкурентов, экономит 6–8 часов.

5. Подготовка материалов для выставок

Презентации стенда, тексты для буклетов, скрипты для менеджеров — публичные материалы.

Пример промпта
Подготовь пакет материалов для выставки [УКАЖИТЕ НАЗВАНИЕ]:

Параметры:
- Отрасль: [УКАЖИТЕ]
- Аудитория: [УКАЖИТЕ]
- Продукт: [УКАЖИТЕ]
- Цель выставки: [УКАЖИТЕ: лиды, бренд, продукт]

Материалы:
1. Слоган стенда (3 варианта, 5-7 слов)
2. Текст для буклета (1 страница A4, структура: проблема-решение-результат)
3. Скрипт для менеджера (приветствие → квалификация → презентация → CTA)
4. Текст для roll-up (2 варианта, 50 слов)
5. Email-фоллоу для встретившихся на стенде

Тон: профессиональный, без агрессивных продаж.

Результат: комплект материалов, экономит 4–5 часов.

6. Локализация маркетинговых материалов

Адаптация контента под разные рынки — язык, культура, регуляторика.

Пример промпта
Адаптируй маркетинговый текст под локальный рынок:

Исходный текст (русский):
[ВСТАВЬТЕ ТЕКСТ]

Целевой рынок: [УКАЖИТЕ СТРАНУ]
Язык: [УКАЖИТЕ ЯЗЫК]

Адаптируй:
1. Перевод с учётом отраслевой терминологии
2. Культурные нюансы (что работает на этом рынке)
3. Регуляторные ограничения (что нельзя обещать)
4. Примеры и кейсы (заменить на локальные, если возможно)
5. CTA (призыв к действию, актуальный для рынка)

Формат: таблица "исходник → адаптация → комментарий"

Результат: локализованный текст, экономит 3–4 часа.

7. Анализ отраслевых трендов

Суммаризация публичной аналитики, новостей, отчётов аналитиков.

Пример промпта (веб-поиск)
Проанализируй тренды отрасли [УКАЖИТЕ] за 2025–2026:

Источники (публичные):
- Отраслевые СМИ
- Отчёты аналитиков (McKinsey, BCG, Deloitte)
- Новости крупных игроков
- Публикации на LinkedIn от экспертов

Вопросы:
1. Ключевые тренды (топ-5)
2. Технологические изменения
3. Регуляторные сдвиги
4. Изменение поведения клиентов
5. Новые бизнес-модели
6. Риски и возможности для нашей компании
7. Рекомендации по позиционированию

Формат: аналитическая записка (1-2 страницы)

Результат: обзор трендов, экономит 5–6 часов.

8. Ценовая аналитика по биржевым котировкам

Мониторинг публичных биржевых данных для ценовой политики.

Пример промпта (веб-поиск)
Проведи ценовую аналитику по сырьевым рынкам:

Индикаторы:
1. LME Aluminum (3-месячный фьючерс) — динамика за 6 месяцев
2. Baltic Dry Index (BDI) — тренд фрахта
3. Курс USD/RUB (если релевантно)
4. Индекс PMI производственного сектора

Вопросы:
- Какая динамика цен на сырьё?
- Как это влияет на себестоимость [УКАЖИТЕ ПРОДУКТ]?
- Прогноз на ближайший квартал (по аналитикам)
- Рекомендации по ценовой политике (общие, без конкретных цифр)

Формат: аналитический бюллетень

Результат: обзор рынка, экономит 2–3 часа.

Проверка знаний

Вопрос 1: Почему разработка контент-плана — во внешний ИИ?

Потому что конечный продукт (посты, статьи) по определению публичен
Потому что не содержит коммерческих цен
Только если не указывать рекламный бюджет

Вопрос 2: Анализ конкурентов — какой контур?

Внешний ИИ — публичные данные о продуктах и ценах
On-prem — коммерческая разведка
Гибрид — данные снаружи, выводы внутри

Вопрос 3: Можно ли во внешнем ИИ готовить питч для инвесторов?

Да, если это общая структура без финансовых данных
Гибрид — структура и текст во внешнем, финальные цифры в on-prem
Нет, питч всегда содержит инсайдерскую информацию
