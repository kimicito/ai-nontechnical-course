3.3 · Простые AI-агенты без кода

🎯 Без кода · ✅ Интерактивный квиз

🎯 Зачем это бизнесу

Агент — это AI, которому поставили цель, и он сам решает, как её достичь. В отличие от обычного чата, агент может выполнять действия: искать в интернете, отправлять письма, записывать в таблицы.

Пример: "Найди 3 поставщика упаковки в Москве, сравни цены, отправь мне таблицу" — агент сам выполняет все шаги.

📚 Что изучаем

Что такое AI-агент

Обычный чат: Вы задаёте вопрос → AI отвечает → Конец.

Агент: Вы ставите цель → AI планирует шаги → Выполняет → Проверяет → Докладывает результат.

3 типа простых агентов

🔍 Исследователь

Ищет информацию, анализирует, составляет отчёт.

Пример задачи
"Проанализируй рынок поставщиков картонной упаковки в Москве.
Найди 5 компаний, сравни цены, сроки, минимальный заказ.
Составь таблицу и рекомендацию."

📊 Аналитик

Обрабатывает данные, находит закономерности, прогнозирует.

Пример задачи
"Проанализируй отчёт о продажах за квартал.
Найди тренды, аномалии, топ-5 товаров.
Спрогнозируй продажи на следующий квартал."

🤝 Ассистент

Помогает в коммуникациях: черновики писем, ответы, напоминания.

Пример задачи
"Подготовь ответ поставщику по его коммерческому предложению.
Вежливо откажи, объясни почему (цена выше рынка на 15%).
Предложи встречные условия."

Пример: агент для закупщика

Задача агенту
"Найди 3 поставщика [товара] в [регионе].
Для каждого:
1. Найди сайт и контакты
2. Определи ценовую категорию
3. Проверь отзывы
4. Сравни с текущим поставщиком

Выведи таблицу: Поставщик | Цена | Сроки | Рейтинг | Рекомендация"

Агент сам:

  1. Поищет в интернете
  2. Посетит сайты
  3. Найдёт отзывы
  4. Сравнит данные
  5. Составит таблицу

👨‍💻 Как делегировать разработчику

Шаблон задачи

"Нужен AI-агент для [задача].

Цель агента: [что должен достичь]

Инструменты, которые нужны агенту:
- [поиск в интернете / email / Excel / 1С / etc.]

Ограничения:
- [не тратить больше $5 на запрос]
- [не делать более 10 шагов]
- [останавливаться для подтверждения при [условие]]

Пример задачи: [привести]
Ожидаемый результат: [привести]

Срок: [дата]"

✅ Чеклист принятия результата

  • Агент достигает цели? (решает задачу от начала до конца)
  • Не зацикливается? (не делает одно и то же по кругу)
  • Укладывается в бюджет? (API-запросы не дороже лимита)
  • Результат проверяем? (можно сверить факты)
  • Есть "человек в цикле" для критичных решений?
  • Ошибки обрабатывает корректно? (не падает, а сообщает)

🎯 Главное из урока

Агент = AI с целью. Он сам планирует шаги, ищет информацию, выполняет действия. Начните с простых задач: исследование, сравнение, анализ. Всегда оставляйте "человека в цикле" для проверки. Это следующий уровень после промптов и автоматизации.
