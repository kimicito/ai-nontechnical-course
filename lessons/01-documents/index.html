1.4 · Работа с документами: анализ, суммаризация, сравнение

🎯 Без кода · ✅ Интерактивный квиз

⚠️ Примеры в этом уроке приведены для обучения. Для загрузки договоров, смет, актов и других конфиденциальных документов необходимо использовать GENAI (корпоративное решение с защитой данных), а не открытые облачные сервисы (ChatGPT, Claude, Kimi).

🎯 Зачем это бизнесу

Менеджер читает в среднем 2-3 часа документов в день: договоры, отчёты, коммерческие предложения, акты, протоколы.

AI сокращает это до 15-20 минут:

  • Суммаризация договора: 30 мин → 2 мин
  • Сравнение 2 КП: 1 час → 5 мин
  • Анализ рисков: 2 часа → 10 мин

Итого экономия: 3-4 часа в день

📚 Что изучаем

3 типа задач с документами

1. Суммаризация (сжатие)

Пересказать суть документа в нескольких пунктах.

💡 Пример промпта
Ты — юрист.
Перескажи договор в 5 пунктах:
1. Что покупаем
2. За сколько
3. Сроки поставки
4. Условия оплаты
5. Штрафы и неустойка

Если пункт не указан в договоре — напиши "Не указано".

Экономия: 30 мин → 2 минуты

2. Сравнение

Сравнить 2 или более документов по заданным критериям.

💡 Пример промпта
Ты — закупщик.
Сравни 2 коммерческих предложения по критериям:
| Критерий | Предложение 1 | Предложение 2 | Лучшее |
| Цена за ед. | | | |
| Мин. заказ | | | |
| Срок доставки | | | |
| Условия оплаты | | | |
| Гарантия | | | |

Выдели различия жирным. Дай рекомендацию.

Экономия: 1 час → 5 минут

3. Анализ рисков

Найти потенциальные проблемы в документе.

💡 Пример промпта
Ты — корпоративный юрист с 15-летним стажем.
Проанализируй договор поставки на риски.

Для каждого риска укажи:
- Описание
- Уровень: Высокий / Средний / Низкий
- Рекомендация по минимизации

Проверь разделы:
☐ Неустойка
☐ Форс-мажор
☐ Расторжение
☐ Порядок разрешения споров
☐ Передача прав третьим лицам

Экономия: 2 часа → 10 минут

🛠️ Как применить

Сценарий: анализ договора

Шаг 1: Загрузите договор в AI (ChatGPT Plus, Claude Pro, или Kimi поддерживают файлы)

Шаг 2: Задайте роль и задачу

Полный промпт
Ты — юрист по корпоративному праву.
Проанализируй приложенный договор поставки.

Задачи:
1. Выдели ключевые условия (цена, сроки, объём, оплата)
2. Найди риски для покупателя
3. Оцени каждый риск: Высокий / Средний / Низкий
4. Дай рекомендации по минимизации рисков
5. Укажи, какие пункты необычны или отличаются от стандарта

Формат: таблица рисков + общий вывод о приемлемости договора.

Шаг 3: Проверьте результат (особенно оценки рисков)

Шаг 4: Сохраните шаблон промпта для будущих договоров

Сценарий: сравнение КП

Задача: Выбрать поставщика из 3 коммерческих предложений.

Промпт
Ты — опытный закупщик.
Сравни 3 коммерческих предложения во вложениях.

Критерии сравнения:
1. Цена за единицу (руб.)
2. Минимальный заказ (шт.)
3. Срок изготовления (дней)
4. Срок доставки (дней)
5. Условия оплаты
6. Наличие сертификатов
7. Гарантийный срок

Выведи таблицей. Для каждого критерия укажи лидера.
В конце — однозначную рекомендацию с обоснованием.

Ограничение: если срок > 14 дней — отметь как риск.

👨‍💻 Как делегировать разработчику

Шаблон задачи

"Нужна система автоматического анализа договоров.

Требования:
1. Загрузка документов: PDF, DOCX, TXT
2. Автоматический анализ по шаблонам:
   - Суммаризация (5 ключевых пунктов)
   - Сравнение с шаблоном компании
   - Оценка рисков (таблица)
3. Сохранение результатов в [Notion/SharePoint/1С]
4. Уведомление ответственному лицу
5. История анализов (поиск по контрагенту)

Пример входного и выходного документа (прикрепить).

Интеграции: [email / 1С / CRM]
Срок: [дата]"

✅ Чеклист принятия результата

  • Все ключевые пункты договора учтены?
  • Риски оценены корректно? (проверьте 2-3 пункта)
  • Рекомендации применимы? (можно реализовать)
  • Формат удобен для передачи коллегам?
  • Нет галлюцинаций? (факты соответствуют документу)
  • Для критичных договоров — проверил юрист?

🎯 Главное из урока

AI анализирует документы за минуты: суммирует, сравнивает, находит риски. Но всегда проверяйте критичные выводы. AI — помощник, не замена юристу. Экономия: 3-4 часа в день на чтение документов.
