📖 Glossary

Complex AI concepts explained in simple words. If you encounter an unfamiliar word in the course — look it up here.

🔤 Key Terms

API

«Data socket» — a way for programs to connect to each other

If site A wants to get data from program B, it uses an API. For example, when a currency calculator gets the current dollar rate from the Central Bank website.

Open-source

Software with open source code — can be downloaded and used for free

Developers publish the code publicly. Anyone can download, study, and modify it. Examples: n8n, LibreOffice. But: support and training may require payment (like Red Hat).

Self-hosted

«On your own server» — the program runs inside the company, not in the cloud

Data never leaves the office. Requires a server and an IT specialist to set up. The license is free, but hardware and maintenance are not.

VPN

Virtual Private Network — a secure tunnel for remote work

Allows an employee working from home to securely connect to the corporate network. For on-premise AI in the office, a local network is needed; for remote work — a VPN.

Firewall

Network barrier — protection from unwanted connections

Like a security guard at the entrance: only lets through those who have permission. Configured by the IT department.

Cron / Schedule

Running a task on schedule: every day, every hour, etc.

For example: "Every Monday at 9:00 check supplier prices." In n8n this is called Schedule, in programming — cron.

JSON

Data format for programs — "a table in text form"

AI often outputs responses in JSON format, which is then converted to Excel/spreadsheets. For humans, it looks like text with brackets.

CSV

Table format — opens in Excel

A simple text format where data is separated by commas. Any spreadsheet can be saved as CSV.

LLM (Large Language Model)

Large language model — the "brain" of AI that understands and generates text

ChatGPT, Claude, Kimi — all are LLMs. Trained on a huge amount of text to understand questions and provide meaningful answers.

Token

A unit of text measurement for AI — approximately ¾ of a word

AI counts not words, but tokens. 1000 tokens ≈ 750 words. The cost of a request depends on the number of tokens.

RAG (Retrieval-Augmented Generation)

Search + generation — AI first finds facts, then answers

Instead of relying only on memory, AI searches for information in your documents and provides an answer based on them. Reduces the risk of fabrications.

Vector Database

Storage where AI searches by meaning, not by exact words

If you search for "expensive supplier", it will also find "high price" and "not cheap." It understands context, not just phrasing.

Prompt

A request or instruction for AI — what you write in the chat

The more precise the prompt, the better the answer. A good prompt contains: role, task, format, constraints, examples.

AI Agent

A program that performs tasks independently according to given instructions

Not just answering questions, but acting: searching, comparing, sending emails, filling out documents.

Hallucination

When AI confidently provides fabricated information

May invent a non-existent law, a fake quote, or a false fact. Always verify critical data.

Fine-tuning

Training a model on your data for specific tasks

A ready-made model is "adjusted" to your documents, terminology, and style. More expensive than regular use, but more accurate.

Edge Cases

Extreme, non-standard situations — when everything goes off-plan

For example: budget = 0, empty file, supplier not responding. Good automation accounts for these scenarios too.

MCP (Model Context Protocol)

Standard for connecting AI to external systems

Allows an AI agent to access your databases, CRM, and email. Like a universal connector for different devices.

On-premise

«On premises» — everything works inside the company, not in the cloud

Data stays within your walls. Secure, but more expensive to maintain. Suitable for confidential information.

Workflow

A sequence of automation steps

For example: receive email → extract data → write to spreadsheet → send notification. In n8n and Zapier, workflows are created visually.

Webhook

«Signal» from one program to another — instant notification

When an event occurs (for example, a new order), system A instantly notifies system B. Without constant polling.

⚙️ Parameters, Automation and Audit

Context Window

«Model memory» — how much text AI remembers at once

If the document is longer than the window, the model forgets the beginning. For analyzing a 50-page contract, you need an LLM with a large window (200K+ tokens) or chunking technique.

Temperature

«Creativity level» of AI — from strict to imaginative

0.1 — predictable, almost deterministic (for contracts, reports). 0.7 — creative, generates variants (for brainstorming, marketing). 1.0+ — may start making things up.

Embedding

«Numeric fingerprint of meaning» — how AI understands that two texts are similar

Each piece of text is assigned a set of numbers (vector). AI finds relevant documents by "mathematical proximity", even if the words are completely different.

Chunking

«Cutting into pieces» — splitting a long document into parts

A 100-page contract is cut into paragraphs of 500 tokens each, so each piece fits into the model's memory. Then AI finds the needed pieces and assembles the answer.

Parsing

«Extracting structure from chaos» — turning unstructured data into a table

Extracting prices, SKUs, units of measure from a supplier's PDF catalog into Excel. Or extracting dates, amounts, and addresses from email correspondence.

SaaS

Subscription software over the internet — no installation, just pay and use

ChatGPT, n8n Cloud, Zapier — all are SaaS. Access via browser, data on provider's servers. Convenient, but for confidential documents you need an on-premise alternative.

Scoring

Point-based evaluation — objective supplier selection or quality check

A supplier scores 85/100 — passes, 60/100 — filtered out. Criteria: price, timelines, reputation, certificates, experience. AI can calculate automatically using a given scale.

Compliance

Compliance with rules — laws, standards, internal regulations

Checking that a contract doesn't violate laws, that an estimate complies with standards, that procurement has passed all approvals. AI cross-checks the document against a requirements checklist.

SLA

Service Level Agreement — response time, availability, penalties

"Response to request — within 2 hours, availability 99.9%, downtime penalty — 1% of monthly fee." AI can monitor SLA compliance and flag violations.

Pipeline

«Conveyor» — a chain of data processing steps

Step 1: receive email → Step 2: extract data → Step 3: write to CRM → Step 4: send notification. In n8n and Zapier, pipelines are built visually, without programming.

🏷️ Course Abbreviations

GENAi

Internal company service based on OpenWebUI

Runs on your servers (on-premise). Secure for confidential documents. Use GENAi for: contracts, reports, personal data, internal protocols.

NDA

Non-Disclosure Agreement — confidentiality contract

Signed with AI providers so they don't use your data for training their models.

ROI

Return on Investment — payback of investments

Shows how much money AI implementation brings compared to costs. ROI = (Profit − Costs) / Costs × 100%.

CRM

Customer Relationship Management system

Stores information about customers, deals, and contacts. Examples: amoCRM, Bitrix24, Salesforce.

ERP

Enterprise Resource Planning system

Unifies accounting, production, procurement, and sales. Examples: 1C, SAP, Oracle.

🤖 Agent Systems and Corporate Tools

OpenClaw

Open-source framework for running AI agents on your own server

Allows you to create a digital employee that works in Telegram, Email, or a corporate messenger. Data never leaves the company — the agent works on-premise. Uses configuration files (SOUL.md, SKILL.md, HEARTBEAT.md) to define behavior.

Hermes Agent

Self-learning AI agent from Nous Research

An agent that creates new skills (SKILL.md) from its own experience and remembers user preferences. Improves with every task, like an experienced employee.

GoClaw

Enterprise platform for agent teams with data isolation

Allows different departments to have their own agents with different access rights. For example, the accounting agent cannot see engineering data. Suitable for large manufacturing companies.

SOUL.md

The "soul" of the agent — a file defining its character and role

For example: "You are an experienced process engineer. Speak concisely, use SCADA data. Don't make assumptions without facts." This is like a job description for a digital employee.

SKILL.md

Skill file — a step-by-step instruction for a typical agent task

For example: "How to analyze a failure log: 1) Load data from SCADA, 2) Group by equipment, 3) Identify recurring failures, 4) Provide recommendations." The agent executes such a skill like a standard operating procedure.

HEARTBEAT.md

Schedule of periodic agent checks

Defines what the agent does automatically and on what schedule. For example: every day at 7:30 check the mining plan vs actual, and if deviation >5% — send a summary. Similar to a duty roster and morning briefing.

MEMORY.md

The agent's long-term memory — facts, decisions, lessons

The agent writes important facts here so it doesn't forget between sessions. For example: "Furnace #3 produces defects at temperatures >1200°" or "Well #47 is unstable at pressure <15 atm." This is like the knowledge base of an experienced employee.

AGENTS.md / USER.md / TOOLS.md

System agent files — rules, user profile, and tools

AGENTS.md — general rules for all agents (code of conduct). USER.md — user profile: who they are, what they do, what can be done without asking. TOOLS.md — list of available tools: SAP, 1C, Grafana, Email.

Corporate messenger

Internal communication tool for the company

Analogous to Telegram/Slack, but deployed inside the company. Agents can send notifications, reports, and alerts to corporate messenger channels — data never leaves the perimeter.

Process control system (SCADA)

Automated system for managing technological processes

A system that collects data from sensors and equipment in real time: temperature, pressure, flow rate, equipment status. A data source for analytics agents and predictive maintenance.

On-premise (on-prem)

«On premises» — programs and data inside the company, not in the cloud

Critical for production data: smelting parameters, mining plans, failure logs. On-prem agents and models run on corporate servers — data is not transmitted to external providers.

Vector Search

Search by meaning, not by exact words

AI turns text into "points" in multi-dimensional space. Words with similar meanings ("penalty", "fine", "forfeit") end up close together. When you ask a question, AI finds the nearest points — even if the wording differs. Important: you cannot "enable" vector search with a prompt command — it works automatically when documents are uploaded, or requires RAG system setup by the IT department.
