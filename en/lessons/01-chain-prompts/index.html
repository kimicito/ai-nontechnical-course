1.3 · Request chains: from simple to complex

🎯 No code · ✅ Interactive quiz

🎯 Why does business need this?

Complex tasks can't be solved in one go.

Example: Supplier market analysis includes: search → comparison → risk assessment → recommendation.

Saving: A chain of 4 prompts = 30 minutes.

📚 What we are studying

What is a request chain

Chain is a sequence of prompts, where each next one uses the result of the previous one.

🛠️ How to apply

Rule: one prompt = one task

❌ Incorrect:"Find suppliers, compare them, assess risks and recommend the best"

✅ Correct: Split into 4 separate prompts

Why: AI gives a superficial answer when asking for too much at once.

When to use chains

SignExample
Task requires > 3 actionsMarket analysis + comparison + recommendation
Each step depends on the previous oneCan't compare until you've collected data
We need an intermediate resultTable to check before analysis
One prompt gives too long answerThe answer is cut off, details are lost

Business chain templates

Chain "Tender"

  1. Gather requirements for the supplier
  2. Make a list of evaluation criteria
  3. Collect proposals (or find suppliers)
  4. Compare by criteria
  5. Assess risks
  6. Formulate a recommendation

Chain "Report"

  1. Collect data for the period
  2. Calculate key metrics
  3. Identify trends and anomalies
  4. Compare with previous period
  5. Formulate conclusions and recommendations

Chain "Agreement"

  1. Analyze key terms
  2. Find risks for each side
  3. Compare with template/previous contracts
  4. Formulate comments
  5. Prepare a list of questions for the counterparty

👨‍💻 How to delegate to a developer

Task template

“We need to automate the analysis chain [process].

✅ Checklist for accepting the result

  • Does each step have a clear input and output task?
  • Is it possible to check the result of the intermediate step?
  • If step 2 is incorrect, is it possible to restart only that step?
  • Is the final result built on all previous steps?
  • Are there no steps that can be combined without losing quality?
  • Does the chain cover the entire task from start to finish?
