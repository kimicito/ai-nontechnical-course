⚡ LEVEL 3 · AUTOMATION

3.1 · No-code automation: n8n and Zapier

🎯 No code · ✅ Interactive quiz

🎯 Why does business need this?

Managers spend 30% of the time on routine: copying data between systems, sending similar letters, filling out reports.

No-code platforms allow you to automate these processes without programming.

Example:"When a letter arrives from a supplier → AI analyzes → if there is a price change → send a notification to the buyer in Telegram"

📚 What we are studying

How automation works

Any automation consists of 3 parts:

  1. Trigger— what starts the process?
  2. Action- what to do?
  3. Condition— when to do it?

n8n - open-source automation

🔗 n8n

💰 Free (self-hosted), cloud from $20/month

n8n — open-source platform with a visual workflow editor. fair-code(free for yourself, paid for cloud SaaS providers).

Key Features
  • 400+ integrations“out of the box” (Google Sheets, Telegram, OpenAI, Notion, CRM)
  • Visual editor— drag the nodes, connect them with arrows
  • Self-hosted— data does not leave your server (confidentiality)
  • HTTP Request— connection to any API without programming
  • JavaScript/Python- you can embed code for complex logic
  • Webhook— accepts data from any systems
  • Schedule— scheduled launch (every hour, day, crown expression)
Example workflow in n8n
1
Trigger: Webhook (a new lead has arrived from the site)
2
AI: OpenAI node - analyzes the request, determines the category
3
Condition: If category = "Purchase" → send to the buyer in Telegram
4
Condition: If category = "Sale" → create a deal in CRM
5
Action: Write log to Google Sheets

Zapier - simplicity and speed

⚡ Zapier

💰 Free up to 100 tasks/month, then from $19/month

Zapier is the most mature tool on the market.5000+ integrations, has been operating since 2011.

Key Features
  • Zaps— automations consisting of a trigger and actions
  • Paths— branching logic (if/else within one Zap)
  • Filter— conditions under which Zap is triggered
  • Formatter— data conversion (dates, numbers, text)
  • Schedule- start by time
  • Webhooks— receiving data from external systems
  • AI Actions- built-in integration with OpenAI, Anthropic, Cohere
  • Tables— built-in database (alternative to Google Sheets)
Example Zap
1
Trigger: Gmail - new email with PDF attachment
2
AI: Zapier AI - extracts key data from PDF (amount, date, counterparty)
3
Filter: If the amount is > 100,000 ₽
4
Action: Send a notification to corporate messenger → #purchases
5
Action: Save data to Google Sheets

When to choose what

n8n If:

  • You work with confidential data (contracts, personal data)
  • There is an IT specialist who can install the server
  • Requires complex logic (loops, conditions, code)
  • Want to save on a subscription (self-hosted = free license, but requires a server and an IT specialist)

Zapier If:

  • Needs to be launched in 10 minutes without server setup
  • You use rare services (you need maximum compatibility)
  • No IT resources - everything should work out of the box
  • Are you willing to pay for convenience and stability?

🛠️ How to apply

Scenario 1: Automated order processing

Problem: Applications from clients come by email, and the manager manually transfers them to the CRM.

Automation in n8n
1
Email Trigger: New letter to zakazy@company.ru
2
OpenAI: Extract: product, budget, deadline, contact
3
Code: Check if all fields are filled in
4
CRM: Create a deal (if fields are filled in)
5
Telegram: Notify manager
6
If: Budget > 100,000 ₽ → notify the director

Saving:2 hours a day to process applications

Scenario 2: Monitoring supplier prices

Automation in Zapier
1
Schedule: Weekly, for example on Mondays 9:00
2
HTTP: Get a price list from the supplier’s website (API or parsing)
3
AI: Compare with last week (Zapier AI or OpenAI)
4
Filter: If the price has changed > 5%
5
Sheets: Write a report in Google Sheets
6
Corporate messenger: Send a notification to the buyer

Scenario 3: Weekly report

Automation in n8n
1
By schedule (cron): Every Friday 17:00
2
Database: Collect data from 1C / CRM (SQL query or API)
3
Code: Aggregate data (grouping, sums)
4
OpenAI: Generate a text report with findings
5
Email: Send to manager
6
Google Drive: Save PDF to archive
