1.2 · How to get AI responses in the right format

🎯 No code · ✅ Interactive quiz

🎯 Why business needs this

By default, AI responds with plain text. You get a paragraph that you have to manually retype into Excel or Word.

Problem: A manager receives supplier analysis as text and spends 15 minutes transferring it to a table.

Solution: Ask AI to output the result as a table right away. Savings: 15 minutes × 20 tasks = 5 hours per week.

📚 What we learn

4 response formats for managers

📊 Table

For comparisons, analysis, reports. Copies to Excel in one click.

☑️ Checklist

For audits, inspections, action plans. Clear and visual.

📧 Ready Email

AI writes an email to a supplier, client, or management immediately.

📋 Step-by-Step Plan

Clear sequence of actions with dates and responsible persons.

How to ask for a table

💡 Prompt example
You are a financial analyst.
Analyze 3 investment projects (data below).

Output the result as a table:
| Project | Payback | Risk | Recommendation |

After the table — brief justification.

How to ask for a checklist

💡 Prompt example
You are an auditor.
Check the supply contract for risks.

Output the result as a checklist:
☐ Risk: [description] — Level: [High/Medium/Low] — Action: [recommendation]

At the end: overall conclusion — sign or not sign.

🛠️ How to apply

Scenario 1: Supplier comparison

Task: Choose a packaging supplier from 3 options.

Prompt
You are an FMCG buyer.
Compare 3 proposals by criteria and output as a table:
| Criterion | Supplier A | Supplier B | Supplier C | Leader |
| Unit price | | | | |
| Min. order | | | | |
| Delivery time | | | | |
| Payment terms | | | | |

Give a recommendation: whom to choose and why.

Result: Ready table for management presentation. Copies to Excel in one click.

Scenario 2: Contract audit

Task: Check a contract for risks before signing.

Prompt
You are a corporate lawyer.
Analyze the supply contract for risks.

Output the result as a checklist:
☐ Risk: [description] — Level: [High/Medium/Low] — Recommendation: [what to do]

At the end: overall conclusion — is the contract acceptable.

Scenario 3: Writing an email

Task: Write an email to a supplier with a quality complaint.

Prompt
You are a procurement manager.
Write a business email to the supplier.

Context: the latest batch has 15% defects, exceeding the acceptable 3%.

Format: ready email in business style.
Structure:
1. Reference to contract and acceptable defect percentage
2. Facts (batch, date, quantity, photos attached)
3. Demand: replacement or refund
4. Response deadline: 5 business days

Tone: firm but polite. No threats.

Result: Ready email that can be sent immediately. Just insert the details.

Scenario 4: Action plan

Task: Create an implementation plan for a new process.

Prompt
You are a project manager.
Create a CRM system implementation plan.

Format: step-by-step plan with dates:
1. Step — Deadline — Responsible — Result
2. Step — Deadline — Responsible — Result
...

Total: 8-10 steps.
Implementation period: 3 months.
Budget: up to $5,000.

Result: Ready plan for management approval. Copies to Word or Excel.

👨‍💻 How to delegate to a developer

Task template

"We need a tool that automatically turns AI responses into Excel tables.

Requirements:
1. Paste AI text — get a ready .xlsx table
2. Works with any tables (supplier comparison, reports, plans)
3. Preserves formatting (bold headers, colors)
4. Simple interface: copy-paste — done
5. No programming required from the user

Example input (attach):
[insert AI response example]

Expected result (attach):
[insert desired Excel]

Deadline: [date]"

✅ Acceptance checklist

  • Format matches the task? (table / list / email / plan)
  • All data present? (no gaps)
  • Can be copied to Excel / Word / Outlook?
  • Structure clear without explanations?
  • Units of measurement specified? (currency, days, percentages)
  • Checked 2-3 random items for accuracy?
