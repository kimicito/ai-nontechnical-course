4.1 · What is an AI agent: from chat to autonomy

🎯 No code · ✅ Interactive quiz

🎯 Why does business need this?

Regular AI is waiting for your commands.The agent acts on his own: sets goals, plans steps, implements, checks results.

Example: You tell the agent, “Prepare a report on the packaging market.”

  1. Searches for information on the Internet
  2. Analyzes what is found
  3. Creates a comparison table
  4. Writes conclusions and recommendations
  5. Sends you the finished document

You spent:5 minutes to set the problem.Agent spent:20 minutes to complete.

🛠️ Prompt vs Skill vs Subagent: how to choose the right tool

Not everything needs to be complex. A one-off task — a single prompt. A recurring process — a contract or skill. A clean context needed — subagent.

MechanismWhen to useExample
PromptOne-off task in the current conversation"Compare 3 supplier quotes"
Project instructionA rule for almost all work in a project"Always write briefly, no fluff"
SkillA repeatable process with boundaries and checks"Every Friday check the contract report"
SubagentNeed a clean context or parallel work"Independently check this contract while I do something else"

Quick decision route

  1. One answer in one conversation? → Prompt
  2. A rule for the whole project? → Project instruction
  3. A repeatable process with a start and exit? → Skill
  4. Need a separate performer or clean context? → Subagent
💡 Evolution example

Week 1: "Analyze this report" — Prompt (one-off)

Week 3: "Check the report every Friday per this contract" — Skill (recurring)

Week 5: "Check the report independently of the main task" — Subagent (isolation)

💡 Decision rule

Start with the lightest mechanism that provides the required repeatability. One-off task — prompt. Constant norm — project instruction. Repeatable process — skill. Need isolation — subagent.

📚 What we are studying

Chatbot vs Agent

💬 Chatbot

  • Waiting for a question
  • Answers once
  • Doesn't remember context
  • Doesn't perform actions
  • Requires constant attention

🤖 Agent

  • Sets goals himself
  • Performs a chain of actions
  • Remembers the state
  • Uses tools
  • Works autonomously

Agent Components

ComponentWhat doesAnalogy
Brain (LLM)Thinks, plans, makes decisionsEmployee head
MemoryRemembers context, history, factsEmployee notepad
ToolsPerforms actions: search, email, ExcelEmployee's hands
SchedulerDecides what to do nextProject Manager

🛠️ How to apply

Examples of business agents

🕵️ Agent-market analyst

Target: Weekly competitor report

Actions: Looks for competitors' prices → Analyzes → Creates a table → Sends a report

Saving:4 hours per week

📋 Assistant buyer agent

Target: Processing incoming CP

Actions: Receives a letter → Extracts data → Compares with the database → Forms a recommendation → Sends to the buyer

Saving:2 hours a day

🔍 Agent-auditor

Target: Checking contracts for compliance with standards

Actions: Receives a contract → Compares with a template → Finds deviations → Generates a report → Sends to a lawyer

Saving:3 hours per document

⛏️ Mining control agent

Target: Daily verification of mining plan vs actual

Actions: Gets data from process control system (SCADA) → Compares with plan → Identifies deviations → Analyzes causes → Sends summary to corporate messenger

Saving: 2 hours per day on manual reconciliation

👨‍💻 How to delegate to a developer

Task template

"We need an AI agent for [task].

✅ Checklist for accepting the result

  • Does the agent achieve the goal completely?
  • Doesn't get stuck on one step?
  • Uses tools correctly?
  • Remembers context between steps?
  • Is it within budget and time?
  • Is there a "man in the loop" for critical decisions?
