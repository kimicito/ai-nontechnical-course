16.2 External Tasks: External AI

⏱ 20 min · for compliance officers
Open Data (External AI)
All tasks in this lesson — public sanctions lists, regulatory acts, open frameworks, standard policies. You can use Claude, ChatGPT, Gemini.

What Can Be Done in External AI

External AI is safe where work is with the regulatory field as such — what lists exist, what the regulator requires, how the methodology works — before applying it to a real name or deal.

Main Rule: if the request doesn't contain specific names, company names, amounts, account numbers, deal dates — you can use external AI.

1. Sanctions Regime Monitoring

Summarizing changes in OFAC, EU, UK, UN and national regulator sanctions lists.

Prompt Example (External AI + Web Search)
Analyze sanctions regime changes for last month:

Jurisdictions: OFAC (USA), EU, UK, UN

Provide:
1. New persons/organizations added to lists
2. Changes in sectoral sanctions
3. New goods/technology restrictions
4. Regulator guidance updates
5. Trends (what to watch)
6. Recommendations for companies in general

Format: analytical note (1-2 pages)

Result: changes overview, saves 3–4 hours.

2. Export Control Conclusions (General)

Analysis of export control norms applicability in general terms, without reference to specific deal.

Prompt Example
Explain export control applicability:

Topic: [SPECIFY: dual-use goods/technology restrictions/military end-use]
Jurisdictions: [SPECIFY: US/EU/CN]

Provide:
1. Applicable export control regimes
2. Goods/technology categories
3. Licensing requirements
4. Exceptions and exemptions
5. License application procedure
6. Practical recommendations (general)
7. Regulatory references

Important: without reference to specific company deal.

Result: export control reference, saves 2–3 hours.

3. Compliance Policy Development Based on Standard Frameworks

Creating policies based on international standards (ISO 37001, UK Bribery Act, FCPA).

Prompt Example
Develop compliance policy based on international standards:

Policy type: [SPECIFY: anti-corruption/AML/sanctions/conflict of interest]
Base framework: [SPECIFY: ISO 37001/UK Bribery Act/FCPA guidance]

Provide:
1. Policy structure (sections)
2. Main provisions (principles)
3. Employee obligations (by level)
4. Documentation procedures
5. Violation investigation procedure
6. Sanctions for violations
7. Review periodicity
8. Appendices (forms, checklists)

Format: ready document for company adaptation.

Result: policy draft, saves 4–5 hours.

4. Business Ethics Code

Developing general part of code based on best practices.

Prompt Example
Develop business ethics code:

Company: large manufacturing company (B2B)
Geography: international operations

Provide:
1. Company values (5-7 items)
2. Behavior in counterparty relations
3. Gifts and hospitality rules
4. Conflict of interest (definition, examples)
5. Asset and information protection
6. Work ethics (discrimination, harassment)
7. Obligation to report violations
8. Investigation procedure
9. Responsibility

Tone: clear, specific examples, no abstractions.

Result: ready code, saves 3–4 hours.

5. Compliance Training Programs

Preparing general training materials.

Prompt Example
Develop compliance training course:

Topic: [SPECIFY: AML/anti-corruption/sanctions/conflict of interest]
Audience: [SPECIFY: all employees/managers/finance department]
Format: 10 slides + test (5 questions)

Draft:
1. Training objectives
2. Key concepts (simple language)
3. Employee obligations
4. Red flags (examples)
5. What to do when suspected
6. Test (5 multiple-choice questions)
7. Self-check checklist

Tone: accessible, with real-life examples.

Result: ready training material, saves 3–4 hours.

6. Industry Compliance Practice Benchmarking

Comparing compliance practices with international holdings based on public data.

Prompt Example (Web Search)
Conduct compliance practice benchmarking:

Industry: [SPECIFY]
Parameters:
- Compliance function structure
- Technology usage (screening automation)
- Training program (hours per employee)
- Whistleblower channels
- External audits (frequency)
- Public reporting (ESG/anti-bribery reports)

Provide:
1. Overview of 3-5 international companies
2. Comparison table
3. Best practices
4. Improvement recommendations
5. Compliance function trends

Sources: public reports, ESG disclosures, ethics codes.

Result: industry practices overview, saves 4–5 hours.

On-prem — policy is internal
