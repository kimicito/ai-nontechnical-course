1.5 Practice: Automate your first task

🎯 Practice · ✅ No code

🎯 Why does business need this?

Theory without practice is useless.Choose one real problem from your work and automate it in 1 hour.

Target: After this lesson you will have working prompt for a real task that saves 2-3 hours per week.

📚 Step 1: Select a task

Ideal task for automation

  • ✅ Repeated > 3 times a week
  • ✅ Occupies > 20 minutes every time
  • ✅ Yes clear algorithm(you can describe the steps)
  • ✅ Does not require critical decisions(safe to be wrong)

Examples of tasks by role

🛒 Buyer

"Compare 3 commercial offers and make a recommendation"

“Check the contract for risks before signing”

"Create a weekly purchasing report from CSV"

📊 Analyst

"Analyze monthly sales and find anomalies"

"Make a forecast for the next quarter based on trends"

⚖️ Lawyer

"Check 5 contracts for typical risks"

"Compare the new agreement with the corporate template"

📦 Logistician

"Optimize your delivery route to 10 points"

"Calculate shipping costs using different methods"

🛠️ Step 2: Automation algorithm

Step 1: Describe the task

What do you do, why, how often, how long does it take.

Example
Task: Comparison of suppliers' CPs

Step 2: Write a prompt (RCOF)

Role, Context, Purpose, Format.

Example
You are an experienced FMCG buyer.

Step 3: Test with 3 examples

Take 3 real examples and check if the prompt works.

What to check
  • Does the result match the request?
  • Is the format convenient?
  • No errors/hallucinations?
  • Has time been shortened?

Step 4: Record the process

Create a template in Notion/Word/Excel.

Template structure
Task name: [what are you doing]

Step 5: Share with colleagues

Show how you save time.

How to share
  • Send a prompt template to the corporate chat
  • Show in a meeting: “I reduced time by X number of hours.”
  • Create a knowledge base with templates for your team

📝Checking understanding

Question 1: Which task is suitable for AI automation?

A) A one-time task that we do once a year
B) Repeats >3 times a week, takes >20 minutes, there is a clear algorithm
C) Creative task without clear criteria

Question 2: What is RCOF in a prompt?

A) Model name
B) Role, Context, Purpose, Format - prompt structure
C) File size in bytes

Question 3: How many examples are needed to test a prompt?

A) At least 3 real examples
B) 1 example is enough
C) No need to check, AI is always right

📝 Practical task

Challenge: Automate one task in 60 minutes

  1. Minutes 0-10: Select a task from the list above (or your own)
  2. Minutes 10-25: Write a prompt using the RCOF formula
  3. Minutes 25-40: Check with 3 real examples
  4. 40-50 minutes: Improve the prompt (what doesn't work?)
  5. 50-60 minutes: Record the template and save

Acceptance criteria

  • Prompt contains RCOF
  • Tested on 3 examples
  • The result is stable (does not change from request to request)
  • Time savings > 50%
  • The template is recorded and available for reuse

The best way to learn is do.
